# templates

このディレクトリは、仮想組織の主要アウトプットテンプレート集です。

目的:
- 出力形式のブレを減らす
- report / handoff / verification / reconciliation を速くする
- 新しい専用エージェントを追加しても、同じ型で運用できるようにする

## 自律改善ループ
- templates は wording / structure / required fields の改善対象に含める
- 既存 agent の出力ブレは、まず templates の改善で吸収する
- 新規テンプレ追加時は、対象 agent と使用場面を必ず明示する
