# 指定濫用防止医薬品 クイックリファレンス

**施行**: 2026年5月1日 | **対象**: 薬剤師・登録販売者

---

## 🎯 5秒で判断

```
購入希望 → 年齢確認 → 他店購入確認 → 購入量確認 → 目的確認 → 販売判断
```

---

## ⚠️ 8成分（必ず確認）

| 成分 | 主な製品 |
|------|----------|
| エフェドリン | 風邪薬、気管支拡張薬 |
| コデイン | 鎮咳薬、鎮痛薬 |
| ジヒドロコデイン | 鎮咳薬 |
| ブロモバレリル尿素 | 睡眠改善薬 |
| プソイドエフェドリン | 風邪薬（鼻づまり） |
| メチルエフェドリン | 気管支拡張薬 |
| **デキストロメトルファン** | 鎮咳薬【新規2026/5】 |
| **ジフェンヒドラミン** | 抗ヒスタミン・睡眠改善薬【新規2026/5】 |

---

## 🚫 18歳未満への販売

| 種類 | 上限 |
|------|------|
| 一般 | 5日分以下・1包装のみ |
| 風邪薬・解熱鎮痛・鼻炎内服 | 7日分以下・1包装のみ |

**大容量・複数個は不可**

---

## ✅ 必須確認（5項目）

1. [ ] **年齢** - 18歳未満は制限あり
2. [ ] **他店購入** - 同日他店舗での購入有無
3. [ ] **購入量** - 複数個・大容量は理由確認
4. [ ] **使用目的** - 適正な使用目的か
5. [ ] **様子** - 濫用の兆候がないか

---

## 🔔 警告サイン（販売要注意）

- 複数店舗を巡っている
- 大容量を繰り返し購入
- 使用目的が曖昧
- 様子が不自然（落ち着きがない等）
- 同伴者と不自然な会話

---

## 💬 声掛け例

**複数個購入時**:
> 「複数お求めですね。どのような症状でお困りですか？」

**若年者の購入**:
> 「年齢確認させてください。18歳未満の方は数量制限があります。」

**濫用が疑われる場合**:
> 「最近、お悩み事はありませんか？相談できる場所もありますよ。」
