# 残薬確認フィードバックシート

**作成日**: 2026年3月7日
**目的**: 医師の残薬確認義務化に伴う薬剤師からの情報提供標準化

---

## 背景

2026年度改定で在宅時医学総合管理料等の算定要件に追加:
> 患家における残薬の状況を患者又はその家族から聴取し、その状況に応じて適切な服薬管理及び処方内容の調整を行うこと。

薬剤師が詳細な残薬情報を文書で提供することで、医師の算定要件支援と処方最適化に貢献。

### 残薬総量の影響評価

| 項目 | 評価 |
|------|------|
| 残薬総日数 | 日分 |
| 在庫過多リスク | □低 □中 □高 |
| 服用不履行リスク | □低 □中 □高 |
| 医師への報告必要性 | □不要 □必要 □緊急 |

**判定基準**:
- 低: 残薬7日分以下、服薬状況良好
- 中: 残薬8〜14日分、または服薬不履行あり
- 高: 残薬15日分以上、または服薬不履行が継続的
