# 指定濫用防止医薬品 自薬局在庫突合チェックリスト

**作成日**: 2026年3月18日
**施行日**: 2026年5月1日（残り43日）
**担当**: 薬剤師エージェント
**目的**: 8成分に該当する自薬局取扱い製品の棚卸しと販売規制区分・記録要件の整理。口頭説明スクリプトとの連携マッピング付き。

---

## 1. 在庫突合の進め方

1. 以下の成分別リストに沿って、自薬局の在庫を確認（☐にチェック）
2. 在庫あり品目の陳列位置・棚札を確認
3. 販売規制区分・記録要件を品目ごとに確認
4. スクリプト連携列を参照し、スタッフ教育時に対応する説明ポイントを把握

## 4. スクリプトとの連携マッピング（スタッフ教育用）

口頭説明スクリプト（`outputs/designated-abuse-prevention-staff-script.md`）の各セクションと、在庫品目の対応関係。

| スクリプト セクション | 対応する在庫品目カテゴリ | 教育時の強調ポイント |
|---------------------|----------------------|-------------------|
| ①導入 | — | 全品目が対象であることの認識 |
| ②8成分の説明 | 全8成分の在庫品目 | 特に成分7・8の新規追加品を実物で確認 |
| ③販売時の対応フロー | 全品目共通 | 第1類（成分1-3）は薬剤師対面必須、指定第2類も記録義務化 |
| ④販売を控えるべき場面 | 特に成分2-4,7,8（乱用リスク高） | コデイン系・睡眠改善薬・DXM含有製品の頻回購入に注意 |
| ⑤記録義務 | 全品目共通 | レジ横の記録様式で全販売を記録 |
| ⑥まとめ | — | 棚札確認→記録様式設置→実践 |
