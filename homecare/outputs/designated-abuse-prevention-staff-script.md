# 指定濫用防止医薬品 スタッフ口頭説明用スクリプト（5分版）

**作成日**: 2026年3月18日
**施行日**: 2026年5月1日
**対象**: 薬局スタッフ朝礼・ミーティング等で使用
**所要時間**: 約5分

---

## 説明者用メモ

このスクリプトは、指定濫用防止医薬品制度について、スタッフに口頭で5分以内に要点を伝えるためのものです。太字部分を特に強調して伝えてください。

---

## スクリプト本文

### ① 導入（30秒）

「5月1日から、**指定濫用防止医薬品**の販売ルールが変わります。大きなポイントは**対象成分が6成分から8成分に増える**ことと、**販売時の確認・記録が義務化**されることです。今日はこの制度のポイントと、実際の対応方法をお伝えします。」

---

### ② 対象の8成分（1分）

「対象となる8成分を確認します。」

**既存6成分**（変更なし）:

1. エフェドリン — 気管支拡張・鼻閉改善
2. コデイン — 鎮咳
3. ジヒドロコデイン — 鎮咳
4. ブロモバレリル尿素 — 鎮静
5. プソイドエフェドリン — 鼻閉改善
6. メチルエフェドリン — 気管支拡張

**新規追加2成分**（ここが重要）:

7. **デキストロメトルファン** — 鎮咳（ベンザブロック・ルル等に含有）
8. **ジフェンヒドラミン** — 抗アレルギー・催眠（ドリエル・レスタミン等に含有）

「特に7番と8番は、**風邪薬や睡眠改善薬に広く含まれている**ので、対象製品が大幅に増えます。棚札で対象製品を確認しておいてください。」

---

### ③ 販売時の対応フロー（1分30秒）

「販売時は次の4ステップで対応します。」

**ステップ1: 年齢確認**
「まず年齢を確認します。**18歳未満の方には、小容量（5日分以下、風邪薬は7日分以下）しか販売できません。**大容量や複数個は不可です。」

**ステップ2: 他店舗・他製品の購入確認**
「他のお店で同じような薬を買っていないか確認します。声かけ例：**『本日、他のお店で同じような成分のお薬をお買い求めになりましたか？』**」

**ステップ3: 購入量と目的の確認**
「複数個や大容量の場合は理由を確認します。声かけ例：**『おまとめ買いですが、ご家族用でしょうか？症状はどのくらい続いていますか？』**」

**ステップ4: 情報提供**
「適正使用について説明します。声かけ例：**『この薬には乱用のおそれのある成分が含まれています。用法・用量を守ってお使いください。症状が続く場合は受診をお勧めします。』**」

---

### ④ 販売を控えるべき場面（1分）

「次のような場合は、販売を控える判断が必要です。」

**販売を控える3つのサイン:**

1. **同じ薬を頻繁に購入している**（週に何度も来る）
2. **大量購入の理由が不自然**（あいまいな説明、話をそらす）
3. **購入者の様子がおかしい**（落ち着きがない、焦燥感がある）

「このような場合は無理に販売せず、**『お体のことが心配ですので、一度かかりつけの先生にご相談されてはいかがでしょうか』**と受診を勧めてください。対応に迷ったら管理薬剤師に相談してください。」

「私たちは**ゲートキーパー**としての役割も担っています。乱用の兆候に気づいたら、地域の保健所や医療機関への紹介も検討します。」

---

### ⑤ 記録義務（30秒）

「販売時は記録が必要です。記録項目は以下の通り：」

- 販売日時
- 購入者氏名・年齢
- 販売製品名・数量
- 他店舗購入状況
- 大容量購入時の理由
- 特記事項（気になる点があれば）

「記録様式は所定の用紙を使用します。レジ横に配置しますので、販売のたびに記入してください。」

---

### ⑥ まとめ（30秒）

「まとめます。5月1日から覚えていただきたいのは3つです。」

1. **対象が8成分に拡大** → 風邪薬・睡眠改善薬も多数該当
2. **販売時は必ず年齢・他店購入・購入量を確認** → 声かけを忘れずに
3. **不審な購入には販売を控える判断も必要** → 迷ったら管理薬剤師へ

「4月中に対象製品の棚札確認と記録様式の準備を完了します。質問があればいつでも聞いてください。」

## 補足資料（説明者用）

### よくある質問への回答

| 質問 | 回答 |
|------|------|
| 常連さんでも毎回確認？ | はい。全購入者に同様の確認が必要です |
| 処方箋の薬も対象？ | いいえ。OTC（一般用医薬品）のみが対象です |
| お客さんが怒ったら？ | 制度上の義務であることを説明し、管理薬剤師に引き継ぐ |
| ネット販売は？ | テレビ電話販売のみ可。薬剤師・登録販売者の対面確認が必要 |

### 声かけ文言まとめ

| 場面 | 文言例 |
|------|--------|
| 年齢確認 | 「お年をお伺いしてもよろしいでしょうか」 |
| 他店購入確認 | 「本日、他のお店で同じような成分のお薬をお買い求めになりましたか？」 |
| 購入量確認 | 「おまとめ買いですが、どなたがお使いですか？」 |
| 情報提供 | 「この薬には乱用のおそれのある成分が含まれています。用法・用量を守ってお使いください」 |
| 受診勧奨 | 「お体のことが心配ですので、一度かかりつけの先生にご相談されてはいかがでしょうか」 |
| 拒否対応 | 「法令で定められた確認義務がございますので、ご理解のほどお願いいたします」 |
