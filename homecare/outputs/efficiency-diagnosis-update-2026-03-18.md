### 3. localStorage対応（v2バージョニング）

**前回改善点への直接対応**。

```javascript
const STORAGE_KEY = 'v2_homecare_efficiency';
const OLD_KEYS = ['homecare_efficiency', 'v1_homecare_efficiency'];

// 旧バージョンキーが存在する場合はリセット
OLD_KEYS.forEach(function(key) {
    if (localStorage.getItem(key) !== null) {
        localStorage.removeItem(key);
    }
});
```

**動作**:
- 旧バージョンキー（homecare_efficiency・v1_homecare_efficiency）を自動削除
- 新バージョンキー `v2_homecare_efficiency` でスコアを保存（回答状態を維持）
- ページ再訪時に前回の回答を復元してボタン選択状態を再現
