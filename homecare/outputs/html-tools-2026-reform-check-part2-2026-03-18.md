### 重症患者の定義 整合性チェック

HTMLに記載されている重症患者の定義（5項目）と2026年改定基準との照合：

| HTML記載 | 公式基準 | 判定 |
|---------|---------|------|
| 在宅時医学総合管理料の対象患者 | 別表第8の2の対象疾患（末期悪性腫瘍・指定難病等）または別表第8の3 | ✅ 概ね妥当 |
| 訪問看護ステーションと連携している患者 | 別表第8の3「月4回以上の訪問看護を受けている者」 | ⚠️ 表現が不正確（月4回以上の条件が抜けている） |
| がん末期・終末期の患者 | 別表第8の2「末期の悪性腫瘍」 | ✅ |
| **要介護度4-5の患者** | 別表第8の3「**要介護3以上**」 | ❌ **不正確（要介護3が含まれない）** |
| 中心静脈栄養・在宅酸素療法等の医療機器使用患者 | 別表第8の2に列挙（在宅酸素療法・在宅中心静脈栄養法等） | ✅ |

### 発見事項

**問題①【中】重症患者定義に誤りあり — 要介護度の基準**
- 現状: 「要介護度4-5の患者」
- 正確な基準: 「要介護3以上」（別表第8の3・包括的支援加算対象）
- 影響: 要介護3の患者が重症患者にカウントされないため、割合が過少評価される
- 修正提案: 「要介護度4-5」→「要介護3以上（別表第8の3）」

**問題②【低】訪問看護連携の条件が曖昧**
- 現状: 「訪問看護ステーションと連携している患者」
- 正確な基準: 「月4回以上の訪問看護を受けている患者」（別表第8の3）
- 影響: 月1-3回の訪問看護患者が誤って重症患者にカウントされる可能性
- 修正提案: 「訪問看護ステーションから月4回以上訪問を受けている患者」

**問題③【情報】加算点数の記載なし**
- 「在宅薬学総合体制加算1=30点」はこのツールには記載なし
- このチェックシートは「重症患者割合の管理ツール」として設計されているため、加算点数を追記するかは設計方針次第
- 追記するとすれば criteriaInfo ボックス内に「※ 重症患者割合20%以上で在宅薬学総合体制加算2（月2回算定）の算定要件を満たす」程度の注記が有用

**問題④【確認】加算の対象の整理**
- このツールが「在宅薬学総合体制加算1」の要件確認ツールと位置づけられているが、重症患者割合20%以上は主に**加算2（月2回算定）の要件**
- 加算1の主要要件は「訪問実績48回/年以上」（点数: 30点）
- ツールの目的が加算1の要件確認であれば、訪問回数の集計機能を持つべき
- ただし「重症患者割合2割要件の管理」ツールとして単独で価値あり

### まとめ

| 判定 | 内容 |
|------|------|
| ✅ 正確な記載 | 20%閾値・JavaScript計算ロジック・がん末期・医療機器使用患者 |
| ❌ 修正必要 | 「要介護度4-5」→「要介護3以上」（**修正推奨**） |
| ⚠️ 要検討 | 訪問看護条件の明確化・加算目的の整理 |

### 算定要件 整合性チェック

公式算定要件（2026年改定）とチェックリスト内容との照合：

| 公式算定要件 | チェックリスト内の記載 | 判定 |
|------------|-------------------|------|
| 在宅時医学総合管理料を算定する医師と同時訪問 | item1「医師と訪問日程・時間を調整済み」/ item5「医師と同時に患者宅へ入室」 | △ 実務的には記載あり、法的要件としての明示なし |
| 他の保険薬局で在宅患者訪問薬剤管理指導料等を算定 | item13「同時指導料の算定要件を満たしているか確認」（汎用） | ⚠️ 具体的でない |
| 患者の同意取得 | **なし** | ❌ |
| 残薬・服薬状況確認の実施 | item6「服薬状況・アドヒアランスを医師と共有」 | ✅ |
| 副作用早期兆候把握 | item7「副作用・有害事象の有無を医師と確認」 | ✅ |
| 算定頻度: 6月に1回 | **記載なし** | ❌ |

### 発見事項

**問題①【中】患者の同意確認がチェックリストにない**
- 2026年改定の算定要件として「患者の同意取得」が明示されている
- 現在のチェックリストには patient consent に関する項目がない
- 修正提案: 「訪問前準備」カテゴリに「患者（家族含む）から同時訪問・同時指導の同意を取得済み」を追加

**問題②【中】算定頻度「6月に1回」の記載なし**
- 訪問診療薬剤師同時指導料は「6月に1回」の制限がある
- 請求要件確認の section（item13-15）や info-box に明記されていない
- 過誤請求リスクあり（月1回や高頻度で算定しないよう注意が必要）
- 修正提案: info-box に「※ 算定頻度: 6月に1回」を追記

**問題③【低】複数名薬剤管理指導訪問料との混同リスク**
- 2026年改定で「複数名薬剤管理指導訪問料（300点）」も新設（別制度）
- 同時指導料（医師+薬剤師）と複数名訪問（薬剤師+補助者）は別物
- info-box に「医師との同時訪問（訪問診療薬剤師同時指導料）は医師との同行が必須」と明記するとより明確

**問題④【確認】算定対象の保険薬局の記載**
- 「他の保険薬局で在宅患者訪問薬剤管理指導料等を算定」という要件が具体的でない
- これは「医師が在宅時医学総合管理料を算定している患者」に対して、「その患者に訪問薬剤管理指導料を算定している薬局」の薬剤師が同時訪問すること
- チェックリストitem2「患者の薬剤情報・服用状況を事前確認」では不十分

### まとめ

| 判定 | 内容 |
|------|------|
| ✅ 正確な記載 | 算定点数300点・医師との同時訪問要件・服薬状況・副作用確認 |
| ❌ 修正必要 | 「患者の同意確認」チェック項目の追加・「6月に1回」算定頻度の明記（**修正推奨**） |
| ⚠️ 要検討 | 算定要件（在宅時医学総合管理料算定医師との同時訪問）の法的文言での明示 |
