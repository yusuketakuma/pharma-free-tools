## カード2: アモキサピン（アモキサン）

### 基本情報

| 項目 | 内容 |
|------|------|
| 成分名 | アモキサピン |
| 商品名 | アモキサンカプセル10mg/25mg/50mg |
| 薬効分類 | 三環系抗うつ薬（第二世代） |
| 削除区分 | **成分単位削除**（同一成分の後発品もなくなる） |
| 製造元 | ファイザー |
| 対応期限 | 経過措置2027年3月末（ただし早期対応推奨） |
| 在宅影響 | **高** — 精神科訪問患者で長期服用中の方は処方変更が必須 |

### 代替薬

| 代替薬 | 分類 | 在宅適性 | 注意点 |
|--------|------|----------|--------|
| **ノルトリプチリン（ノリトレン）** | 三環系 | ○ | 同系統で切替しやすい。抗コリン作用注意 |
| **クロミプラミン（アナフラニール）** | 三環系 | ○ | 錠剤あり。鎮静作用がやや強い |
| **デュロキセチン（サインバルタ）** | SNRI | ○ | 1日1回。腎・肝障害時は禁忌に注意 |
| **ミルタザピン（リフレックス/レメロン）** | NaSSA | ◎ | OD錠あり。食欲増進作用（低体重患者に有利） |

### 対応フロー

```
☐ Step 1: MCS患者抽出 → アモキサピン処方中の在宅患者を特定
☐ Step 2: 主治医（精神科医）に相談 — 処方変更の可否・代替薬の選定
       ※ 精神科薬は主治医判断が必須。薬剤師からの一方的な変更提案は避ける
☐ Step 3: 主治医の指示に基づき代替薬を手配
☐ Step 4: 患者・家族への説明（服用方法の変更点・副作用の違い）
☐ Step 5: 切替後2〜4週間のフォローアップ（副作用・効果確認）
```

### 特記事項
- 精神科領域の処方変更は**主治医の判断が最優先**。薬剤師は情報提供に徹する
- 長期服用中の患者は急な中止で離脱症状のリスクあり → 漸減切替を推奨
- 認知症合併の高齢在宅患者では抗コリン作用の少ない薬剤（SNRI/NaSSA）を推奨

## カード3: リラグルチド（ビクトーザ）

### 基本情報

| 項目 | 内容 |
|------|------|
| 成分名 | リラグルチド（遺伝子組換え） |
| 商品名 | ビクトーザ皮下注 18mg/3mL 1キット |
| 薬効分類 | GLP-1受容体作動薬（2型糖尿病治療薬） |
| 削除区分 | **成分単位削除** |
| 製造元 | ノボノルディスク |
| 対応期限 | 経過措置2027年3月末 |
| 在宅影響 | **高** — 糖尿病の在宅患者で毎日自己注射中の方に直接影響 |

### 代替薬

| 代替薬 | 投与頻度 | 在宅適性 | 備考 |
|--------|----------|----------|------|
| **デュラグルチド（トリュリシティ）** | **週1回** | ◎ | オートインジェクター。自己注射が簡便 |
| **セマグルチド（オゼンピック）** | **週1回** | ◎ | 週1回。同クラスGLP-1 |
| **セマグルチド（リベルサス）** | **毎日経口** | ◎ | 注射不要の経口GLP-1。服用条件あり（空腹時） |
| **チルゼパチド（マンジャロ）** | **週1回** | ○ | GIP/GLP-1デュアル。血糖降下作用が強力 |

### 対応フロー

```
☐ Step 1: MCS患者抽出 → ビクトーザ処方中の在宅患者を特定
☐ Step 2: 主治医に情報提供 — 同クラス代替薬リスト＋患者の腎機能・HbA1c情報
☐ Step 3: 代替薬決定後、手配＋デバイス教育準備
       ※ 週1回製剤への変更の場合、投与曜日の設定が必要
☐ Step 4: 患者・家族への説明（デバイスの使い方・投与スケジュール変更）
☐ Step 5: 切替後1〜2ヶ月のHbA1c・低血糖モニタリング
```

### 特記事項
- 週1回製剤への切替は患者の自己注射負担軽減になる可能性あり（ポジティブに提案可）
- リベルサス（経口）は注射嫌いの患者に有利だが、服用条件（起床時空腹・水120mL以下・30分飲食禁止）が在宅高齢者には難しい場合あり
- 腎機能（eGFR）確認必須 — GLP-1製剤は腎機能低下時の用量調整が異なる

### 基本情報

| 商品名 | レミニール錠4mg/8mg/12mg、OD錠、内用液 |
| 薬効分類 | コリンエステラーゼ阻害薬（アルツハイマー型認知症治療薬） |
| 削除区分 | 経過措置（ただし12mg規格は供給停止情報あり） |
| 製造元 | ヤンセンファーマ |
| 対応期限 | 2027年3月末（早期確認推奨） |
| 在宅影響 | 中 — 認知症の在宅患者で維持量12mg服用中の方に影響 |

### 代替薬

| 代替薬 | 剤形 | 在宅適性 | 備考 |
|--------|------|----------|------|
| **ドネペジル（アリセプト）** | 錠・OD錠・細粒・ゼリー | ◎ | 1日1回。剤形豊富で在宅に最適 |
| **リバスチグミン（イクセロン/リバスタッチ）** | **貼付剤** | ◎ | パッチ剤。嚥下困難者に最適 |
| **メマンチン（メマリー）** | 錠・OD錠 | ○ | NMDA拮抗薬（併用も可） |

### 対応フロー

```
☐ Step 1: MCS患者抽出 → レミニール処方中（特に12mg）の在宅患者を特定
☐ Step 2: 供給状況の最新確認（卸に在庫照会）
☐ Step 3: 12mg供給停止の場合 → 主治医に代替薬提案
       ※ 認知症薬の切替は症状悪化リスクあり。主治医との綿密な相談が必要
☐ Step 4: 切替時は2週間以上かけて漸減・漸増
☐ Step 5: 切替後1ヶ月間の認知機能・BPSD変化を注意観察
```

### 特記事項
- 認知症治療薬の切替は家族の不安が大きい → 丁寧な説明と安心材料の提供が重要
- リバスチグミンパッチは嚥下困難・服薬拒否の在宅患者に有力な選択肢
- ガランタミン4mg/8mg規格が入手可能な場合、維持量の調整で対応できる可能性あり

## カード5: メマンチン（メマリー等）

### 基本情報

| 項目 | 内容 |
|------|------|
| 成分名 | メマンチン塩酸塩 |
| 商品名 | メマリー錠5mg/10mg/20mg、OD錠、ドライシロップ |
| 薬効分類 | NMDA受容体拮抗薬（中等度〜高度アルツハイマー型認知症治療薬） |
| 削除区分 | 経過措置（一部後発品が限定出荷） |
| 対応期限 | 2027年3月末 |
| 在宅影響 | 中 — 先発品メマリーは安定供給。後発品使用中の場合に影響 |

### 代替薬

| 対応方針 | 内容 | 備考 |
|----------|------|------|
| **先発品へ切替** | メマリー（第一三共） | 供給安定。後発品からの切替が最もスムーズ |
| **他の後発品** | メマンチン「サワイ」「トーワ」等 | 供給状況を卸に確認 |
| **ChEI単独療法** | ドネペジル or リバスチグミン | メマンチン併用をやめChEI単独に戻す選択肢 |

### 対応フロー

```
☐ Step 1: MCS患者抽出 → メマンチン後発品を処方中の在宅患者を特定
☐ Step 2: 使用中の後発品メーカーの供給状況を確認
☐ Step 3: 限定出荷の場合 → 先発品（メマリー）への切替を主治医に提案
☐ Step 4: 患者・家族への説明（薬の名前が変わるが同じ成分であること）
☐ Step 5: 切替後の副作用（めまい・頭痛・便秘）モニタリング
```

### 特記事項
- 先発品→先発品の切替なので、患者への影響は最小限
- 後発品使用患者の場合、薬価差に注意（自己負担額が変わる可能性）
- メマンチンOD錠・ドライシロップは嚥下困難の在宅患者に有用

### 基本情報

| 項目 | 内容 |
|------|------|
| 成分名 | トラニラスト |
| 商品名 | レミカットカプセル100mg |
| 薬効分類 | 抗アレルギー薬（ケミカルメディエーター遊離抑制薬） |
| 削除区分 | **成分単位削除** |
| 製造元 | キッセイ薬品 |
| 対応期限 | 経過措置2027年3月末 |
| 在宅影響 | 中 — アレルギー疾患の在宅患者で長期服用中の方に影響 |

### 代替薬

| 代替薬 | 分類 | 在宅適性 | 備考 |
|--------|------|----------|------|
| **フェキソフェナジン（アレグラ）** | 第二世代抗ヒスタミン | ◎ | 眠気少。OTC併用注意 |
| **ロラタジン（クラリチン）** | 第二世代抗ヒスタミン | ◎ | 1日1回。OD錠あり |
| **ビラスチン（ビラノア）** | 第二世代抗ヒスタミン | ○ | 空腹時服用が必要 |
| **モンテルカスト（シングレア/キプレス）** | ロイコトリエン拮抗薬 | ◎ | 喘息合併に有利 |
| **クロモグリク酸Na（インタール）** | 遊離抑制薬 | △ | 吸入・点眼のみ |

### 対応フロー

```
☐ Step 1: MCS患者抽出 → レミカット処方中の在宅患者を特定
☐ Step 2: 適応疾患を確認（気管支喘息・アレルギー性鼻炎・蕁麻疹・ケロイドなど）
       ※ トラニラストはケロイド予防にも使われる — 適応によって代替薬が異なる
☐ Step 3: 適応に応じた代替薬を主治医に提案
☐ Step 4: 患者への説明（特にOTC薬との重複注意）
☐ Step 5: 切替後のアレルギー症状コントロール確認（2〜4週間）
```

### 特記事項
- トラニラストのケロイド予防適応には直接代替薬がない → 主治医（皮膚科・形成外科）と協議
- アレルギー性疾患の場合、第二世代抗ヒスタミン薬への切替は比較的容易
- 高齢在宅患者では眠気の少ない薬剤を優先（フェキソフェナジン・ロラタジン・ビラスチン）
