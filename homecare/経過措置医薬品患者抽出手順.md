# 経過措置医薬品患者抽出手順書

**期限**: 2026年3月31日
**作成日**: 2026年3月7日
**残り日数**: 24日

---

## 背景

2026年3月、経過措置医薬品387品目が削除予定。
そのうち54品目は成分単位で削除されるため、代替薬への切り替えが必要。

---

## 優先成分（訪問患者への影響大）

| 成分名 | 商品名例 | 影響領域 | 優先度 |
|--------|----------|----------|--------|
| リラグルチド | ビクトーザ | 糖尿病 | 高 |
| アモキサピン | アモキサン | 精神科・老年 | 高 |
| トラニラスト | レミカット | アレルギー | 中 |

---

## MedicalCareStation患者抽出クエリ

### 手順

1. **MedicalCareStationを開く**
2. **患者検索 → 薬剤検索タブ**を選択
3. **以下の成分名で検索**:
   ```
   リラグルチド
   アモキサピン
   トラニラスト
   ```
4. **処方中患者リストをエクスポート**
5. **訪問予定患者を優先的に確認**

### SQLクエリ例（参考）
```sql
-- 優先成分（訪問患者への影響大）
SELECT 患者ID, 患者名, 処方薬剤名, 次回訪問日
FROM 処方データ
WHERE 成分名 IN ('リラグルチド', 'アモキサピン', 'トラニラスト')
  AND 処方終了日 IS NULL
ORDER BY 次回訪問日 ASC;
```

## 注意点

- 経過措置期間中は使用可能（使用期限までの猶予あり）
- 患者の状態によっては緊急性が高い場合がある
- 訪問患者は通院患者よりも切り替え調整に時間を要する
