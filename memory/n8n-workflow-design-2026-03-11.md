# n8n自動投稿ワークフロー設計書

## 概要

薬剤師向け無料ツール集の集客・収益化を目的とした、n8nによるRSS→SNS自動投稿ワークフローの設計書。

## 目標

- 月間PV: 10,000→30,000（3倍増）
- 有料製品（AIプロンプト集）流入: 月50件
- 投稿工数: 0分（完全自動化）

## アーキテクチャ

```
[RSS Feed] → [n8n Trigger] → [OpenAI 要約] → [Telegram 承認] → [X/Instagram 投稿]
```

## 詳細設計

### 1. RSS フィードソース

**優先度: 高**
- 薬局新聞 (https://www.yakkyoku-newspaper.com/feed)
- 日経メディカル (https://medical.nikkeibp.co.jp/rss.rdf)
- m3.com 薬局関連 (RSS)

**優先度: 中**
- 厚労省プレスリリース (https://www.mhlw.go.jp/stf/rss.rdf)
- PMDA 安全性情報 (RSS)
- CloseDi 供給情報 (代替手段必要)

**優先度: 低**
- 薬学雑誌 最新号
- JPS 学会情報

### 2. n8n ワークフロー構成

#### Node 1: RSS Read
- タイプ: RSS Feed Read
- 設定: 15分間隔でチェック
- フィルタ: 薬局・在宅医療・調剤報酬関連キーワード

#### Node 2: AI Content Generation
- タイプ: OpenAI Chat Model
- モデル: GPT-4o-mini（コスト最適化）
- プロンプト:
```
あなたは薬剤師向けSNS投稿の専門家です。
以下の記事を基に、薬剤師に役立つX(Twitter)投稿文を作成してください。

記事タイトル: {{title}}
記事内容: {{content}}

要件:
- 140文字以内
- 薬剤師の業務効率化・収益向上に焦点
- 絵文字2-3個使用
- ハッシュタグ3個 (#薬剤師 #調剤報酬 #在宅医療)
- CTA: 無料ツール集への誘導
```

#### Node 3: Human Approval (Telegram)
- タイプ: Telegram Trigger
- 機能: 投稿内容をTelegramで送信→返信で承認/却下
- 承認キーワード: "OK", "投稿"
- 却下キーワード: "NG", "スキップ"

#### Node 4: X (Twitter) Post
- タイプ: X (Twitter) API
- 必要な権限: Write access
- コスト: Basic $100/月（API v2）

#### Node 5: Instagram Post (Optional)
- タイプ: Instagram Graph API
- 要件: 画像生成（DALL-E 3 または Canva API）
- コスト: Instagram API無料

### 4. 代替手段（低コスト版）

**X API費用削減案**
1. IFTTT無料枠: RSS→Twitter（投稿内容制御不可）
2. Mastodon/Bluesky: 無料API（X以外のSNS展開）
3. Buffer無料枠: 手動承認→予約投稿

**最小構成コスト: $25/月 (n8nのみ)**
