## ツール候補

### 優先度1: 薬価計算ツール（後発品切り替えシミュレーター）

**概要**: 先発品から後発品への切り替えでどれくらい薬価が下がるか計算

**機能**:
- 薬品名入力 → 先発品・後発品の薬価表示
- 複数薬品の一括計算
- 月間・年間の節約額表示
- PDF出力（患者説明用）

**技術スタック**:
- フロントエンド: HTML/CSS/JS（Vanilla）
- データ: 薬価マスターCSV（公開データ）
- ホスティング: GitHub Pages

**開発工数**: 8-12時間

---

### 優先度2: G承認チェックリスト生成ツール

**概要**: 包括管理契約（G承認）のチェックリストを自動生成

**機能**:
- 患者情報入力 → チェックリスト自動生成
- PDF出力
- 保存・再編集機能

**技術スタック**:
- フロントエンド: React/Vue（SPA）
- ホスティング: GitHub Pages

**開発工数**: 12-16時間

---

### 優先度3: 薬歴テンプレート生成ツール

**概要**: 患者情報を入力すると薬歴テンプレートを自動生成

**機能**:
- 患者背景入力 → SOAP形式テンプレート生成
- コピー&ペーストでMCSへ貼り付け可能
- テンプレートのカスタマイズ保存

**技術スタック**:
- フロントエンド: HTML/CSS/JS
- AI連携: OpenAI API（オプション）

**開発工数**: 6-10時間
