# 最初の10人の顧客獲得ガイド

## ソース
Reddit r/passive_income調査（2026年3月）- 月$5K-$30Kの20人の創業者へのインタビュー結果

## 核心インサイト

**最初の10人は「信頼」から生まれる**

- Product Hunt、Twitter、広告、Hacker News、コールドメール、インフルエンサー → **全員0件**
- ニッチコミュニティ参加、直接アプローチ、個人ネットワーク → **全員これで成功**

## 方法1: ニッチコミュニティ参加（11/20人が使用）

### ステップ
1. **コミュニティ特定**
   - ターゲット顧客が集まるsubreddit、Facebookグループ、Discord、フォーラムを探す
   - 例: 薬剤師→r/pharmacy、日本薬剤師会Facebook、薬局経営者Discord

2. **信頼構築期間（6週間〜3ヶ月）**
   - 商品リンクを投稿しない
   - 質問に答える、知識を共有する、困っている人を助ける
   - 「何を作っているか」聞かれるまで待つ

3. **自然な紹介タイミング**
   - 誰かが「〇〇で困っている」と言った時
   - 「実はそれを解決するものを作っているんです」と自然に紹介

### 成功事例
- 「請求書に関する質問に6週間答え続けた。『何のツールを使っているの？』と聞かれた時、1時間で4件の申し込み」
- 「Facebookグループで3人の飲食店オーナーのメニュー同期問題を手動で解決。自動化ツールを作っていると伝えたら『どこで支払えばいい？』と言われた」

### 日本語テンプレート
```
【コミュニティ参加→信頼構築投稿例】

初期段階（1-2週目）:
「〇〇について勉強中です。皆さんはどうされていますか？」

知識提供段階（3-4週目）:
「以前同じ問題で悩みました。私の場合は〜という方法で解決しました。参考になれば。」

信頼確立段階（5-6週目）:
「〇〇さんの状況、私も経験あります。具体的には〜というアプローチが効果的でした。」

自然な紹介（タイミングが来たら）:
「実は、その問題を解決するツールを作っているんです。もし試してみたい方がいれば、フィードバックいただけると嬉しいです。」
```

## 方法2: 直接アプローチ（5/20人が使用）

### ステップ
1. **問題を抱える人を探す**
   - Redditコメント、フォーラムスレッド、Twitterの不満投稿、アプリストアレビュー
   - 検索キーワード: 「困っている」「悩み」「使いにくい」「改善したい」

2. **ピッチではなく質問でアプローチ**
   ```
   悪い例: 「私のツールを試してみませんか？〇〇の機能があります。」
   
   良い例: 「〇〇で苦労されているのを見かけました。似たような問題を解決するものを作っているのですが、もしよければ試していただけないでしょうか？役に立つか教えていただけると嬉しいです。」
   ```

3. **30人にアプローチ→8人反応→4人試用→3人有料化**
   - 成功率: 10%（30人中3人）
   - でも最初の3人は14ヶ月後も継続顧客

### 日本語テンプレート
```
【直接アプローチDM例】

件名: 〇〇について

本文:
〇〇さん、こんにちは。

Redditで〇〇についての投稿を拝見しました。私も以前同じ問題で悩んでいたので、お気持ちよくわかります。

実は、その問題を解決するためのツールを開発中です。もしよろしければ、ベータ版を試してみていただけないでしょうか？

もちろん無料です。ただ、実際に使ってみて「役に立つかどうか」教えていただけると、開発の参考になります。

ご興味があれば、こちらからアクセスできます: [URL]

お返事お待ちしています。
```

## 方法3: 個人ネットワーク（3/20人が使用）

### ステップ
1. **身近な問題解決から始める**
   - 家族、友人、元同僚、取引先
   - 「〇〇さん、それ困っていませんか？作ってみましょうか」

2. **紹介の連鎖を作る**
   - 1人が気に入れば、同じ業界の人を3人紹介してくれる
   - 3人が5人紹介→10人到達

### 成功事例
- 「母の会計事務所用に作った。母が他の3人の会計士に伝えた。その3人がさらに5人に伝えた。オンラインに一度も投稿せず10人到達」

### 日本語テンプレート
```
【個人ネットワーク活用チェックリスト】

□ 家族・親戚で同じ問題を抱えている人は？
□ 友人・知人で同じ業界の人は？
□ 元同僚で独立した人は？
□ 取引先で困っている担当者は？
□ 地域のビジネス交流会に参加している人は？

アプローチ例:
「〇〇さん、以前話していた〇〇の件ですが、実はそれを解決するツールを作ってみたんです。よかったら試してみませんか？」
```

## やってはいけないこと（20人中0人が成功）

| 手法 | 理由 |
|------|------|
| Product Hunt | ローンチ時期にはまだ信頼がない |
| Twitter/X | フォロワーがいないと見てもらえない |
| 広告 | 最初の10人には過剰投資 |
| Hacker News | 技術系でないと相手にされない |
| コールドメール | 返信率が低すぎる |
| インフルエンサー | 費用対効果が悪い |
