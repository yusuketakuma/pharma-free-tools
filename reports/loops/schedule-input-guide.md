# 日程管理 — 入力テンプレート

ゆうすけが期日・予定を入力する際に使うテンプレート。
どの形式でも可。1件だけでも構いません。

## 入力例

```
件名: ○○の提出期限
日時: 2026-04-03 17:00
備考: 必須 / 任意 / 準備状況
```

## 入力経路

- Telegram「たまAI」への直接メッセージ
- OpenClaw UIでのセッション送信
- このファイルへの直書き（schedule-input.md）

## 日程管理担当の処理内容

1. 入力を受け取り、構造化（件名 / 日時 / 備考 / 重要度）
2. 他の予定・期日との競合チェック
3. 秘書本部長へ短い報告
4. schedule-latest.md を更新
