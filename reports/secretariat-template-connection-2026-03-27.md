# secretariat template connection

## 実行したこと
- 秘書本部長と配下4担当の agent 定義に、優先して使うテンプレートを明記した。
- regular report に秘書本部セクションを追加し、日常運用で見える形にした。

## 接続した内容
- 秘書本部長 → 直接支援 / 日程管理 / メール整理 / 訪問薬剤支援 / regular report
- 直接支援担当 → 直接支援テンプレ + regular report
- 日程管理担当 → 日程管理テンプレ + regular report
- メール担当 → メール整理テンプレ + regular report
- 訪問薬剤支援担当 → 訪問薬剤支援チェックリスト + regular report

## 期待する改善
- 秘書本部の各担当が、何の型で返すべきか迷いにくくなる
- regular report で秘書本部の状況が見えるようになる
- self-improvement で template 側の required fields を改善しやすくなる

## 次に行うこと
- 必要になった実務から順に秘書本部へ投入し、足りない項目があれば template を育てる
- regular report の本部別セクションを他本部にも広げる
