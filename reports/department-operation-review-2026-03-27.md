# department operation review

## 見直した観点
- 部署内の情報共有
- 本部長の責務
- 本部長から取締役会への報告形式
- 取締役会から本部長経由の部署内指示伝播

## 入れた改善
- 本部長の責務に、shared context 維持・部署内/取締役会案件の仕分け・指示翻訳を追加した
- 部署内 shared context の最低項目を fixed fields 化した
- 本部長→取締役会の報告テンプレを追加した
- 取締役会→本部長→部署内の指示伝播テンプレを追加した
- 本部長→担当への標準 handoff テンプレを追加した
- 部署内情報共有 guideline を追加した

## 期待する改善
- 本部内で priority / blocked / waiting_on / next_action の語彙が揃う
- 本部長が『橋渡し役』として機能しやすくなる
- 取締役会の判断が、部署内で動ける形に翻訳される
- routine と例外が混ざりにくくなる

## 次に行うこと
- 秘書本部から先に、この型で実際の本部運用を試す
- regular report に本部長報告の要約欄を増やす
