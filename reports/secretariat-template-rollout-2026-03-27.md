# secretariat template rollout

## 実行したこと
- 秘書本部向けテンプレートを4種追加した。
- 直接支援、日程管理、メール整理、訪問薬剤支援の4領域を最初の対象にした。
- templates 一覧にも追記し、今後の自律改善対象として扱いやすくした。

## 追加したテンプレートの要点
- 直接支援テンプレ: 今日の優先順位、すぐ対応すること、あとで確認すること、詰まりを分ける
- 日程管理テンプレ: 既存予定、締切、調整必要予定、確認待ちを分ける
- メール整理テンプレ: 優先度、要求内容、推奨処理、下書き要点、人手確認点を分ける
- 訪問薬剤支援チェックリスト: 訪問前、当日、説明事項、記録、未確認事項を分ける

## 期待する改善
- 秘書本部の仕事を少しずつ増やしても、最初から型で受けられる
- direct support / schedule / mail / homecare support の出力ブレが減る
- self-improvement で wording や required fields を育てやすくなる

## 次に行うこと
- 秘書本部長と配下担当の agent 定義に、どのテンプレートを優先利用するか追記する
- regular report に秘書本部セクションを追加する
