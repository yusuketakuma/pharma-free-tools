# Client Communication Automation Starter Kit — 2026-03-27

## 結論
`sidebiz-project-scout` の 3 つの需要シグナルは、別々に追うより **1 つの再利用可能なスターターキット** に束ねるのが最適。  
今回は、**問い合わせ返信 / 口コミ獲得 / 予約フォロー** をまとめた「無料監査 + テンプレートパック」の形で固定する。

## Board review
### Board Visionary
- 3 テーマは根っこが同じで、`client communication` の手作業負担に集約できる。
- 1テーマずつ作るより、監査・テンプレ・追客を束ねた方がレバレッジが高い。

### Board User Advocate
- 使う側は、個別施策を3本読むより、1枚の監査シートで見たい。
- 提案は「今の手作業を減らす」ことに絞った方が分かりやすい。

### Board Operator
- 最小実行案は、**無料監査の質問票 + 返信テンプレ 3 本 + 成果指標 3 つ** をひとまとめにすること。
- まずは営業文面ではなく、再利用できる中身を固定する。

### Board Auditor
- 外部送信や課金、信頼境界の変更はしない。
- これはあくまで内製の提案メモ / テンプレートであり、対外送信は別判断。

### Board Chair
- 候補は 3 つあるが、個別採否ではなく **1 つの bundle として採用** する。
- 目的は「新規案件を増やすこと」ではなく、「次の実験を始めやすい形で残すこと」。

## 今回見つけた候補
1. **採用**: 問い合わせ返信テンプレート
   - 価値: 返信遅延を減らし、最初の離脱を抑える

2. **採用**: 口コミ依頼 / 口コミ返信テンプレート
   - 価値: レビュー獲得の運用負担を下げる

3. **採用**: 予約リマインド / 追客テンプレート
   - 価値: 予約化率と来店率の改善に直結する

## Board の採否判断
- 3 件を個別に回さず、**1 つのスターターキットとして採用**
- 今回は調査止まりにせず、再利用しやすい骨子を残す

## 見送った理由
