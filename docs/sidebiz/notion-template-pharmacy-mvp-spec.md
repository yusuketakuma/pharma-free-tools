## 3. ダッシュボード構成

### 3.1 メインダッシュボード

**レイアウト**: 2カラム

**左カラム**:
1. **今週の訪問カレンダー**（Linked View）
   - ソース: Visit Logs / カレンダービュー
   - フィルター: 今週

2. **未対応タスク**（Linked View）
   - ソース: Tasks / リストビュー
   - フィルター: ステータス ≠ 完了
   - ソート: 期限昇順

**右カラム**:
1. **経過措置医薬品アラート**（Linked View）
   - ソース: Transitions / ボードビュー
   - フィルター: 対応状況 = 未確認 OR 医師相談中

2. **今月の統計**（数式ブロック）
   - 訪問回数: `Visit Logs.filter(今月).count()`
   - 算定点数合計: `Visit Logs.filter(今月).sum(算定点数)`
   - かかりつけ薬剤師対象者数: `Patients.filter(かかりつけ薬剤師 = true).count()`
