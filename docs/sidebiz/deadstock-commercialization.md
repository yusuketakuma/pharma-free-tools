# DeadStockSolution 商用化プラン

## 1. ターゲット顧客

### プライマリ
- 中小規模調剤薬局（1-10店舗）
- 在庫調整コストに課題感がある
- 既存手段: 電話/FAX/メールで他薬局と個別調整

### セカンダリ
- ドラッグストアチェーン
- 地域薬局ネットワーク

## 2. 課題と解決価値

| 課題 | 解決 | 価値（月額換算） |
|------|------|------------------|
| 期限近い在庫の廃棄ロス | 交換先を自動マッチング | 廃棄損 ¥50K-200K/月削減 |
| 交換先探しの工数 | 候補を自動提示・優先度付け | 担当者工数 5-10h/月削減 |
| 調整漏れ・確認遅延 | ステータス一元管理・通知 | 機会損失防止 |

## 3. 料金プラン（案）

### Light（小規模薬局向け）
- 月額: ¥9,800
- 在庫登録: 1,000品目まで
- 提案数: 無制限
- 交換実績: 無制限
- サポート: メール

### Standard（中規模薬局向け）
- 月額: ¥19,800
- 在庫登録: 5,000品目まで
- 提案数: 無制限
- 交換実績: 無制限
- サポート: メール + チャット
- 統計ダッシュボード: 詳細版

### Enterprise（チェーン向け）
- 月額: ¥49,800〜（店舗数応相談）
- 在庫登録: 無制限
- 提案数: 無制限
- 交換実績: 無制限
- サポート: 専任担当 + 電話
- カスタマイズ: 可能
- 導入サポート: 含む
