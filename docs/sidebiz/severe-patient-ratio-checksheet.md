# 64ツール目: 重症患者割合チェックシート

## 概要
2026年調剤報酬改定で新設される「重症患者割合20%要件」の達成状況を可視化するチェックシート

## 背景
- 2026年6月実施の調剤報酬改定で、訪問薬剤管理の重症患者割合20%以上が要件化
- 在宅医療を行う薬局で管理ニーズが高い
- 手動計算は手間がかかり、見落としリスクがある

## 機能
1. 患者リスト入力（重症度フラグ付き）
2. 重症患者割合の自動計算
3. 20%要件の達成状況表示
4. アラート（20%を下回るリスク）
5. localStorage保存

## RICE評価
- R（Reach）= 4: 在宅医療を行う薬局で需要あり
- I（Impact）= 2: 20%要件の可視化で算定漏れ防止
- C（Confidence）= 0.8: 改定対応で確実なニーズ
- E（Effort）= 0.5: シンプルなチェックシート
- **Score = 12.8**

## 実装仕様

### 重症患者の定義（2026年改定）
1. 在宅時医学総合管理料の対象患者
2. 訪問看護ステーションと連携している患者
3. がん末期・終末期の患者
4. 要介護度4-5の患者
5. 中心静脈栄養・在宅酸素療法等の医療機器使用患者

### UI構成
- 患者入力フォーム（名前、重症度フラグ）
- 現在の割合表示（プログレスバー）
- 要件達成状況（✅/⚠️）
- 患者リスト（編集・削除可能）
