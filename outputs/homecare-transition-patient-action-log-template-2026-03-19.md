# Homecare 経過措置患者別対応ログ テンプレ - 2026-03-19

## 目的
MCSで経過措置対象患者を抽出した直後に、**患者ごとの対応優先度・代替候補・説明要点・期限**を同じ形式で記録し、抜け漏れを防ぐためのテンプレ。

---

## 使い方（5分で開始）
1. MCSから対象患者CSVを出力する
2. 優先6成分（テリパラチド / アモキサピン / リラグルチド / ガランタミン / メマンチン / トラニラスト）を先に埋める
3. **A: 即日 / B: 3営業日以内 / C: 今週中 / D: 監視** で優先度を付ける
4. 医師照会・代替薬提案・患者説明の順で進める
5. 完了したら最終列に日付を入れる

---

## 優先度ルール

| 区分 | 意味 | 目安 |
|------|------|------|
| A | 即日対応 | 供給停止・返品期限直近・次回訪問直近 |
| B | 3営業日以内 | 代替薬準備や医師照会が必要 |
| C | 今週中 | 期限はあるが即日は不要 |
| D | 監視継続 | 供給解除/在庫十分/患者影響軽微 |

## 6成分の初動メモ

### 1. テリパラチド
- 優先度: **A**
- 期限: **2026-03-28返品期限**
- 代替候補: テリボン / フォルテオ / オスタバロ
- 先に確認: 在庫本数・ロット・患者別割当て・返品可否

### 2. アモキサピン
- 優先度: B
- 代替候補: ノルトリプチリン / クロミプラミン / デュロキセチン / ミルタザピン
- 注意: 精神科主治医判断が必須

### 3. リラグルチド
- 優先度: B
- 代替候補: トリュリシティ / オゼンピック / リベルサス / マンジャロ
- 注意: 注射頻度・デバイス変更の説明が必要

### 4. ガランタミン
- 優先度: B〜C
- 代替候補: ドネペジル / リバスチグミン / メマンチン
- 注意: 認知機能・BPSD悪化の観察を入れる

### 5. メマンチン
- 優先度: C
- 代替候補: 先発メマリー / 他後発品
- 注意: まず供給状況と剤形を確認

### 6. トラニラスト
- 優先度: C
- 代替候補: フェキソフェナジン / ロラタジン / ビラスチン / モンテルカスト
- 注意: 適応（アレルギー/ケロイド）で分岐する
