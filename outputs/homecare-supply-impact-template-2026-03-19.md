## 2. 在宅影響判定

### 優先確認カテゴリ
- [ ] 注射剤・輸液
- [ ] 鎮痛薬
- [ ] 抗凝固薬
- [ ] 糖尿病薬
- [ ] 認知症/精神科関連
- [ ] その他、代替困難な慢性薬

### 判定観点
- [ ] 継続投与が止まるとリスクが高い
- [ ] 代替薬の選定が難しい
- [ ] 在宅患者の訪問時説明が必要
- [ ] 残薬/在庫/次回配送に影響する
- [ ] 医師照会が必要になる可能性がある

### 影響判定
- [ ] 影響なし
- [ ] 軽微（監視継続）
- [ ] 中等度（患者/医師対応の準備）
- [ ] 高い（当日中の対応要否を判断）

---

## 3. 対応要否

### A. 影響なし
- [ ] 次回確認へ
- [ ] 記録のみ残す

### B. 軽微
- [ ] 在庫確認のみ
- [ ] 次回訪問時の説明候補として控える

### C. 中等度
- [ ] 代替候補の確認
- [ ] 対象患者の有無確認
- [ ] 医師照会の要否検討

### D. 高い
- [ ] 当日中の在庫確認
- [ ] 対象患者の抽出
- [ ] 医師照会/処方提案準備
- [ ] 患者・家族への説明要点整理

## 使い方（次サイクル指示）
1. CloseDi再取得直後にこのテンプレを埋める
2. 差分がなければ「影響なし」で終了し、確認時刻だけ残す
3. 差分があれば在宅影響カテゴリを優先評価
4. trainer報告には「新規差分 / 継続 / 保留」を優先して転記する
