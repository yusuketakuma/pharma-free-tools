# DeadStockSolution 商用化プラン

## サービス概要
中小調剤薬局向けの在庫管理・期限管理SaaS

## ターゲット
- 中小調剤薬局（1-10店舗）
- 訪問薬剤管理を実施する薬局
- 手動管理から脱却したい薬局

## 料金プラン

### Light プラン（¥9,800/月）
**対象**: 小規模薬局（1-2店舗）
**機能**:
- 在庫管理（最大2,000品目）
- 期限アラート
- 基本レポート
- メールサポート

### Standard プラン（¥19,800/月）
**対象**: 中規模薬局（3-5店舗）
**機能**:
- 在庫管理（最大10,000品目）
- 全レポート機能
- API連携
- 優先サポート
- ユーザー無制限

### Enterprise プラン（¥49,800〜/月）
**対象**: 大規模薬局（6店舗〜）
**機能**:
- 在庫管理（無制限）
- カスタムレポート
- 専任サポート
- データ移行支援
- SSO連携

## 競合優位性
1. **薬局特化** - 一般的在庫管理ツールと差別化
2. **訪問薬剤管理対応** - 在宅医療領域の需要増加
3. **日本市場専用** - 調剤報酬・薬事法対応
4. **低価格** - 大手競合（¥50,000〜）より安価

## 販売チャネル
1. **直接販売** - LP経由のトライアル→有料転換
2. **紹介プログラム** - 既存ユーザーからの紹介
3. **業界展示会** - 医療機器展示会出展
4. **コンテンツマーケティング** - note記事・YouTube
