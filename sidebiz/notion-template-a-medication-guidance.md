### ✅ 服薬指導チェックリスト（薬効群別）

#### 【降圧薬】

- [ ] 血圧測定の推奨

- [ ] めまい・ふらつきの注意
- [ ] 塩分制限の説明
- [ ] 定期受診の重要性

#### 【糖尿病薬】

- [ ] 血糖測定の推奨

- [ ] 食事との関係（食前・食後・食直前）
- [ ] 甘いものの携帯推奨
- [ ] 足のケア

#### 【抗凝固薬】

- [ ] 出血症状の説明（鼻血・血尿・血便）
- [ ] 他院受診時の申告
- [ ] 納豆・クロレラ避ける
- [ ] 定期血液検査の重要性
- [ ] 転倒防止

#### 【スタチン】

- [ ] 服用時間（夕食後推奨）
- [ ] 筋肉痛・脱力感の注意
- [ ] 定期肝機能検査

#### 【抗生物質】

- [ ] 全量服用の重要性
- [ ] 服用間隔の厳守
- [ ] 下痢・発疹の注意
- [ ] ヨーグルト・乳酸菌の推奨
