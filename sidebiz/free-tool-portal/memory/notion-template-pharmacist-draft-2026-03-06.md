# Notionテンプレート企画 - 薬剤師向け業務管理システム
## 2026-03-06 22:36 JST

---

## コンセプト

薬剤師の日常業務をNotion一つで管理できるオールインワンテンプレート。
薬歴入力、服薬指導メモ、在庫管理、学習記録を統合。

---

## ターゲット

- 調剤薬局勤務薬剤師
- 在宅医療担当薬剤師
- 管理薬剤師
- 薬学生（実習用）

---

## テンプレート構成（ドラフト）

### データベース1: 患者カルテ

| プロパティ | 型 | 用途 |
|-----------|---|------|
| 患者ID | タイトル | 識別子 |
| 氏名 | テキスト | 氏名 |
| 生年月日 | 日付 | 年齢計算 |
| 処方箋 | 関連 | 処方箋DBへリンク |
| 在宅フラグ | チェックボックス | 在宅患者識別 |
| 備考 | テキスト | アレルギー等 |
| 最終来局 | 日付 | 自動更新 |

### データベース2: 処方箋ログ

| プロパティ | 型 | 用途 |
|-----------|---|------|
| 処方日 | 日付 | 受付日 |
| 患者 | 関連 | 患者DBへリンク |
| 薬品リスト | マルチセレクト | 処方薬 |
| 疑義照会 | チェックボックス | 有無 |
| 薬歴 | テキスト | 入力内容 |
| 担当薬剤師 | ユーザー | 自動記録 |

### データベース3: 在庫管理

| プロパティ | 型 | 用途 |
|-----------|---|------|
| 薬品名 | タイトル | 商品名 |
| 在庫数 | 数値 | 現在庫 |
| 発注点 | 数値 | アラート閾値 |
| メーカー | セレクト | 薬価表参照 |
| 更新日 | 日付 | 自動更新 |

### データベース4: 学習ログ

| プロパティ | 型 | 用途 |
|-----------|---|------|
| タイトル | タイトル | 学習内容 |
| カテゴリ | セレクト | 疾患/薬効/法規等 |
| 日付 | 日付 | 学習日 |
| ソース | URL | 参考元 |
| メモ | テキスト | 要点 |

### データベース5: タスク管理

| プロパティ | 型 | 用途 |
|-----------|---|------|
| タスク | タイトル | 内容 |
| ステータス | セレクト | 未着手/進行中/完了 |
| 優先度 | セレクト | 高/中/低 |
| 期限 | 日付 | 期限日 |
| 関連患者 | 関連 | 患者DBへリンク |
