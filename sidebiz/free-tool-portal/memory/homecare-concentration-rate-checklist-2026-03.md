## 📊 集中率85%の壁

- 全国の薬局の約39.3%が集中率85%超に該当
- 集中率85%超で月間受付600回超 → 調剤基本料2（都市部）
- 基本料1→2転落で **-11.7点** の減収（月1,000枚換算で年間約-140万円）

### 他科処方箋必要枚数の計算式

```
他科処方箋必要枚数(x) ≧ A / 0.85 - (A + B)

A = 特定の医療機関からの受付回数
B = その他の医療機関からの受付回数
```

**例**: メイン病院から月1,800枚の場合
→ 月318枚以上の他科処方箋が必要

## 💡 集中率計算の注意点

### 受付回数にカウントされない
- 時間外等処方箋

### 受付回数に入るが分母に入らない（集中率に影響しない）
- 施設・居住系の在宅
- オンライン服薬指導分

### 受付回数に入り、分母だけに入る（集中率を下げる効果あり）
- 特定の医療機関以外の外来処方
- 個人在宅処方箋

**戦略**: 「受付回数に入り、分母だけに入る」処方箋を積み上げる
