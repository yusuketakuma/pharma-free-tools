# 施策G25: 処方疑義パターンDB（無料Webツール25）

## 概要

処方疑義のパターンを検索・参照できる無料データベースツール。
「この疑義、どう対応する？」を検索で解決。頻出パターン・対応例・照会テンプレートを収録。

## 背景・目的

- **課題**: 処方疑義の対応パターンが属人化。新人薬剤師は毎回悩み、ベテランでも稀なパターンは忘れがち。
- **解決**: 頻出疑義パターンを検索可能なDB化。対応例・照会テンプレートをワンクリックコピー。
- **差別化**: 既存ツール（G7疑義照会メール生成）は「文章生成」だが、G25は「パターン検索」に特化。

## 特徴

### 🔍 検索機能
- キーワード検索（薬剤名・疑義タイプ・対応方法）
- カテゴリフィルタ（用量・相互作用・重複・禁忌・投与期間・在庫・その他）
- 頻度順ソート（よくある疑義パターンを上位表示）

### 📋 パターン収録（初期100パターン目標）
**用量・用法カテゴリ（30パターン）**
- 腎機能低下時の用量調整必要
- 高齢者への用量過多
- 小児用量の確認必要
- 体重あたり用量の計算確認
- etc.

**相互作用カテゴリ（25パターン）**
- ACE阻害薬+ARB併用
- ワルファリン+NSAIDs併用
- MAO阻害薬+セロトニン系薬剤
-etc.

**重複投与カテゴリ（15パターン）**
- 同一成分含有製剤の重複
- 薬効群重複（降圧薬等）
- 一般名+先発品の重複
- etc.

**禁忌・慎重投与カテゴリ（15パターン）**
- 妊婦禁忌薬の確認
- 小児禁忌薬の確認
- アレルギー既往の確認
- etc.

**投与期間・継続カテゴリ（10パターン）**
- 長期投与の妥当性確認
- 抗生物質の投与期間
- ステロイドの漸減確認
- etc.

**その他（5パターン）**
- 在庫不足・代替品提案
- 後発品変更の確認
- etc.


### 📱 UI/UX

- 詳細モーダル（チェックポイント・照会テンプレート）

### 🔧 技術仕様

## 差別化ポイント

1. **既存ツールとの差分**:
   - G7（疑義照会メール生成）: 文章生成特化
   - G25（処方疑義パターンDB）: パターン検索特化
   - → 併用可能（G25でパターン検索 → G7で文章生成）

2. **競合状況**:
   - 処方疑義パターンを検索可能にした無料ツールは皆無
   - 医薬品検索（J-TOCUS等）はあるが、「対応パターン」に特化したものはない

3. **独自価値**:
   - 実務経験に基づく100パターン（拡張可能）
   - 照会テンプレートのワンクリックコピー
   - 関連ツールへのスムーズな誘導
