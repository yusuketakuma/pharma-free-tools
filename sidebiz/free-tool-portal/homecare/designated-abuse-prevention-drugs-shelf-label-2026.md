---

## 棚札A: 18歳未満販売制限

```
┌─────────────────────────────────────────┐
│     【指定濫用防止医薬品】              │
│                                         │
│   18歳未満の方へ                        │
│   ・小容量（1包装）のみ販売可能         │
│   ・5日分以下（一般薬）                 │
│   ・7日分以下（風邪薬・解熱鎮痛薬）     │
│                                         │
│   複数個・大容量の販売はできません      │
└─────────────────────────────────────────┘
```
