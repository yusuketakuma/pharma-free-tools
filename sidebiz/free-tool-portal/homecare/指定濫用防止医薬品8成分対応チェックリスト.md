## 薬剤師対応チェックリスト

### 販売時確認事項
- [ ] 他店舗での購入状況確認
- [ ] 他の指定濫用防止医薬品の購入状況確認
- [ ] 購入者の氏名・年齢確認
- [ ] 購入者の状況（様子など）確認
- [ ] 大容量・複数個購入時の理由確認

### 18歳未満への販売制限
- [ ] 原則: 小容量（5日分以下）の1包装単位のみ販売
- [ ] 風邪薬・解熱鎮痛薬・鼻炎内服薬: 7日分以下の1包装単位のみ
- [ ] 大容量製品の販売不可
- [ ] 複数個の販売不可

### ゲートキーパー対応
- [ ] 濫用が疑われる購入者への声かけ
- [ ] 悩みを抱えた本人への気づき
- [ ] 必要に応じた専門機関への相談促進

### 施行前準備（5月1日まで）
- [ ] 対象製品外箱の注意喚起表示の確認
- [ ] 販売手順書（マニュアル）の更新
- [ ] スタッフへの周知・教育

## 新規追加2成分の詳細

### デキストロメトルファン
- **用途**: 鎮咳薬（せき止め）
- **作用**: 延髄のせき中枢に作用してせきを抑える
- **過量摂取時の症状**: 錯乱、興奮、幻覚などの精神症状

### ジフェンヒドラミン
- **用途**: 抗ヒスタミン薬（アレルギー用薬・睡眠改善薬）
- **作用**: 中枢神経抑制作用
- **過量摂取時の症状**: 意識障害、せん妄、痙攣
