## MedicalCareStation 患者抽出クエリ案（2026-03-07 5:30 AM作成）

### 抽出条件（優先度高: 成分削除54品目のうちhomecare関連）

```sql
-- パターン1: 成分名での検索（汎用）
SELECT
    p.患者ID,
    p.患者名,
    p.生年月日,
    r.処方日,
    r.薬品名,
    r.成分名,
    r.用法用量
FROM
    患者マスタ p
    JOIN 処方歴 r ON p.患者ID = r.患者ID
WHERE
    r.成分名 IN (
        'リラグルチド',      -- ビクトーザ
        'アモキサピン',      -- アモキサン
        'トラニラスト'       -- レミカット
    )
    AND r.処方日 >= '2026-01-01'
ORDER BY
    r.成分名, p.患者名
```

### 抽出条件（パターン2: 薬品名での検索）

```sql
-- 薬品名部分一致での検索
SELECT
    p.患者ID,
    p.患者名,
    r.処方日,
    r.薬品名
FROM
    患者マスタ p
    JOIN 処方歴 r ON p.患者ID = r.患者ID
WHERE
    (r.薬品名 LIKE '%ビクトーザ%'
     OR r.薬品名 LIKE '%アモキサン%'
     OR r.薬品名 LIKE '%レミカット%')
    AND r.処方日 >= '2026-01-01'
```
