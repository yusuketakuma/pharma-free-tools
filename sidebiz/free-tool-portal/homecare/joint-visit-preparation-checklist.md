# 医師との同時訪問準備チェックリスト

**作成日**: 2026年3月7日
**目的**: 新設「訪問診療薬剤師同時指導料（300点/6ヶ月）」の円滑な算定支援

---

## 算定要件確認

### 基本要件
- [ ] 対象患者が在宅時医学総合管理料を算定中
- [ ] 当該薬局で在宅患者訪問薬剤管理指導料を算定中
- [ ] 患者・家族の同意を得ている
- [ ] 医師との同時訪問を実施

### 算定制限
- **頻度**: 6ヶ月に1回まで
- **対象外**: 施設入居時等医学総合管理料の患者
- **期間**: 退院後1ヶ月以内の訪問は入院基本料に含まれる

---

## 同時訪問前準備チェックリスト

### 1. 患者情報整理（訪問3日前まで）

- [ ] **処方歴確認**
  - 現在の処方薬一覧（薬品名・用量・用法）
  - 最近の処方変更履歴
  - 副作用・相互作用の懸念事項

- [ ] **残薬状況**
  - 前回訪問時の残薬記録
  - 服薬遵守状況
  - 残薬発生理由の分析

- [ ] **臨床データ**
  - 直近の検査値（腎機能・肝機能等）
  - バイタルサインの推移
  - 体重変化

### 2. ポリファーマシー評価（訪問2日前まで）

- [ ] **処方薬の適正性評価**
  - 5種類以上の内服薬がある場合: リスト化
  - 重複作用のある薬剤の有無
  - 不要・減量可能な薬剤の検討

- [ ] **潜在的問題の抽出**
  - [ ] 薬剤重複
  - [ ] 相互作用リスク
  - [ ] 適応外処方の確認
  - [ ] 用量過多/過少
  - [ ] 服薬スケジュールの複雑性

### 3. 医師への事前情報提供（訪問1日前まで）

- [ ] **情報提供書の作成**
  ```
  【同時訪問事前情報】
  患者名:
  訪問予定日:

  ■現在の問題点（優先度順）
  1.
  2.
  3.

  ■提案内容
  -

  ■根拠
  -
  ```

- [ ] **送付方法の確認**
  - FAX / メール / ICT連携ツール

### 4. 訪問当日持ち物

- [ ] 処方経緯一覧表
- [ ] 残薬状況写真・記録
- [ ] 提案資料（処方変更案）
- [ ] 薬剤情報（添付文書・参考資料）
- [ ] 同時訪問記録用紙

---

## 同時訪問時の実施事項

### 訪問中
1. **患者状況の共有**
   - 医師による診察内容の聴取
   - 薬剤師による服薬状況報告

2. **処方検討**
   - ポリファーマシー対策の提案
   - 残薬対策の提案
   - その場での処方変更決定

3. **患者・家族への説明**
   - 変更内容の説明
   - 服薬指導

### 訪問後
- [ ] 同時訪問記録の作成
- [ ] 処方変更の反映確認
- [ ] 次回同時訪問の予定（6ヶ月後）
