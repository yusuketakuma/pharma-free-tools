## 変更点サマリー

| 項目 | 現行（～2026.5） | 改定後（2026.6～） |
|------|----------------|-----------------|
| 加算1 点数 | **15点** | **30点** |
| 訪問実績要件 | 直近1年間 **24回以上** | 直近1年間 **48回以上** |
| 加算2（単一建物1人） | 50点 | **100点** |
| 計算対象 | 在患者訪問薬剤管理指導料の算定件数 | 同左 |

> ⚠️ **要注意**: 実績要件が2倍（24→48回）に引き上げ。現在の算定実績が24～47回の場合、6月以降は加算1が算定不可になる。

## ステップ3: 判定

```
合計訪問回数を確認

  ≥ 48回 → 6月改定後も加算1算定可（届出更新へ）
  24〜47回 → ⚠️ 要注意（加算1不可・加算算定停止の可能性）
  < 24回 → 現行でも要件未充足（届出取り下げ or 未算定を確認）
```

## ステップ4: アクション別対応

### 📗 パターンA: 48回以上達成済み
- [ ] 6月1日付で届出更新（地方厚生局に変更届出書提出）
- [ ] レセコンの加算1算定設定が6月改定点数（30点）になっているか確認
- [ ] 掲示物の更新（在宅薬学総合体制加算1 →「30点」に変更）

### 📙 パターンB: 24〜47回（要件未達）
- [ ] 4月・5月の訪問強化で48回達成が可能か試算（下記）
  - 必要追加訪問回数: 48回 - 現在の回数 = **__回**
  - 残り期間（4〜5月）: 約8週間
  - 週あたり必要追加件数: __回 ÷ 8週 ≒ **__回/週**
- [ ] 達成可能なら4〜5月の訪問頻度・新規患者受け入れを検討
- [ ] 達成困難なら6月より加算1算定停止を予定し、届出取り下げ準備
- [ ] 加算2（100点/件）への移行可能な単一建物1人患者の確認

### 📕 パターンC: 現行も未算定 / 未届出
- [ ] 在宅患者の訪問件数を確認
- [ ] 加算1要件（48回）達成に向けた在宅患者受け入れ戦略を検討
- [ ] 要件達成後に届出

## ステップ5: 届出・更新手続き

### 届出先
- 地方厚生局（各都道府県）
- 様式：保険医療機関及び保険薬局の施設基準に係る届出書（施設基準添付書類）

### 届出タイミング
| 時期 | 内容 |
|------|------|
| **5月中旬までに** | 実績集計・要件確認完了 |
| **5月31日までに** | 届出書類作成・提出（地方厚生局へ郵送 or 持参） |
| **6月1日** | 改定点数（30点）での算定開始 |

> ℹ️ **参考**: 届出書類は地方厚生局のHPでDL可能。様式第6（在宅薬学総合体制加算）を使用。

## 参考: 実績が不足する場合の収益影響

| ケース | 現行月収 | 6月以降 | 変化 |
|--------|---------|---------|------|
| 加算1を毎月10件算定 | 15点×10 = 150点 = 1,500円/月 | 算定停止 → 0円 | ▲1,500円/月 |
| 加算1を毎月20件算定 | 15点×20 = 300点 = 3,000円/月 | 算定停止 → 0円 | ▲3,000円/月 |
| 加算2（新規）月5件 | 50点×5 = 250点 = 2,500円/月 | 100点×5 = 5,000円/月 | +2,500円/月 |

> ✅ **代替算定**: 要件未達でも、単一建物1人患者に対しては「加算2（100点）」が新設されるため収益補完が可能。
