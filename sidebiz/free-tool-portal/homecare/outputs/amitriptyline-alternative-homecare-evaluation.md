# アミトリプチリン代替薬 在宅適用評価シート

**作成日**: 2026年3月18日
**担当**: 薬剤師エージェント
**背景**: サワイ限定出荷 + トリプタノール（日医工）ニトロソアミン問題の二重リスク

---

## 1. 評価の目的

アミトリプチリンの供給不安定化に伴い、在宅患者における代替薬候補の適性を評価し、主治医への提案資料とする。

## 3. 在宅特有の評価ポイント

### 剤形の在宅適性

| 剤形 | 在宅適性 | 理由 |
|------|----------|------|
| OD錠（口腔内崩壊錠） | ◎ | 嚥下困難・簡易懸濁対応 |
| 錠剤（粉砕可能） | ○ | 簡易懸濁法で投与可能 |
| カプセル（開封可） | △ | 開封手技が必要 |
| 腸溶性カプセル | × | 粉砕・開封不可 |
| 液剤 | ◎ | 用量調節容易 |

### 高齢在宅患者での注意事項

- **ポリファーマシー**: 薬物相互作用の確認が最重要
- **転倒リスク**: 鎮静性のある薬剤は骨折リスク増大
- **認知機能**: 抗コリン作用のある薬剤は認知症悪化リスク（アミトリプチリンはこのリスクが高かった）
- **腎機能**: 加齢によるeGFR低下を考慮した用量調整
- **服薬アドヒアランス**: 1日1回製剤が望ましい

## 4. 主治医への問い合わせトークスクリプト

### パターンA: 神経障害性疼痛の場合

```
先生、お世話になっております。在宅訪問薬剤師のゆうすけです。

○○様のアミトリプチリンの件でご相談があります。

現在、アミトリプチリンは2つの問題が重なっています。
サワイ製が品質問題で限定出荷中、日医工のトリプタノールも
ニトロソアミン検出の問題が出ており、安定供給が困難な状況です。

○○様は神経障害性疼痛でアミトリプチリン○mgをお使いですが、
代替薬として以下をご検討いただけないでしょうか。

【第一候補】デュロキセチン（サインバルタ）20mg〜60mg
 → 1日1回服用、高齢者でも転倒リスクが比較的低く、
   腎機能がeGFR30以上であれば使用可能です。

【第二候補】プレガバリン（リリカ）25mg〜150mg
 → エビデンスは豊富ですが、めまい・傾眠に注意が必要です。
   腎機能に応じた用量調節をお願いいたします。

切り替えの方向性について、ご指示いただけますでしょうか。
```

### パターンB: うつ・不安の場合

```
先生、お世話になっております。在宅訪問薬剤師のゆうすけです。

○○様のアミトリプチリンの件でご相談があります。

現在の供給問題（サワイ限定出荷＋トリプタノールニトロソアミン問題）
により、安定的な供給が困難な状況です。

○○様はうつ・不安でアミトリプチリン○mgを少量お使いですが、
代替薬として以下をご検討いただけないでしょうか。

【第一候補】エスシタロプラム（レクサプロ）5mg〜10mg
 → 薬物相互作用が少なく、高齢者の第一選択として推奨されています。

【第二候補】ミルタザピン（リフレックス）7.5mg〜15mg
 → 不眠や食欲低下を合併されている場合に有利です。
   鎮静作用があるため就寝前1回の服用です。

なお、三環系からSSRI/NaSSAへの切り替えは漸減漸増が望ましいですが、
アミトリプチリン少量（10mg程度）であれば、比較的スムーズに
切り替え可能と考えます。ご指示をお願いいたします。
```

### パターンC: 不眠（補助）の場合

```
先生、お世話になっております。在宅訪問薬剤師のゆうすけです。

○○様のアミトリプチリン10mg就寝前の件でご相談があります。

供給問題により継続が困難な状況です。
不眠の補助目的でお使いのため、以下への変更をご検討いただけないでしょうか。

【第一候補】レンボレキサント（デエビゴ）2.5mg〜5mg
 → オレキシン受容体拮抗薬で依存性が低く、
   翌朝の持ち越しも少ないです。

切り替えのタイミングについてご指示をお願いいたします。
```

## 5. 切り替え時のモニタリング項目

| 確認項目 | 確認時期 | 方法 |
|----------|----------|------|
| 症状変化（疼痛・気分・睡眠） | 切替後1週・2週・4週 | 訪問時ヒアリング |
| 副作用出現 | 切替後毎回訪問時 | バイタル確認・問診 |
| 転倒の有無 | 切替後毎回訪問時 | 本人・家族確認 |
| 服薬アドヒアランス | 切替後毎回訪問時 | 残薬確認 |
| 離脱症状 | 切替後1〜2週 | 不眠・不安・発汗等 |
