# テリパラチド現場フォローアップ準備メモ

**所要時間**: 約10分
**作成日**: 2026-03-18 22:22
**担当**: 薬剤師エージェント（homecare-30m-assign）
**目的**: 3/19〜21の3日間アクションプラン実行中のゆうすけ進捗確認用

---

## 概要

テリパラチド「サワイ」返品期限（3/28）まで残り**10日**。ゆうすけが**明日3/19から**3日間アクションプランを実行する。本メモは各日の確認ポイントと対処法を整理したもの。

---

## 1. 3日間スケジュールと確認ポイント

### Day 1: 3/19（水）— 在庫確認

| 項目 | 内容 |
|------|------|
| **アクション** | 薬局内のテリパラチド「サワイ」在庫を確認 |
| **確認時刻** | 3/19 夜（業務終了後） |
| **確認事項** | ✅ 在庫数の把握（箱数・ロット番号）<br>✅ 返品期限の再確認（3/28）<br>✅ 返品対象製品の特定 |
| **想定問題** | ❌ 在庫が予想より多い<br>❌ ロット番号不明で返品対象が判別できない |
| **対処法** | → 3/20の電話で追加対応を相談<br>→ メーカーにロット番号問い合わせ |

### Day 2: 3/20（木）— 電話連絡

| 項目 | 内容 |
|------|------|
| **アクション** | 業者またはメーカーへ返品受付の電話 |
| **確認時刻** | 3/20 夜（電話完了後） |
| **確認事項** | ✅ 返品受付番号の取得<br>✅ 返品方法の確認（FAX/メール/Web）<br>✅ 返品期限の再確認（3/28） |
| **想定問題** | ❌ 電話が繋がらない<br>❌ 担当者が不在で受付できない<br>❌ 返品条件が合わない |
| **対処法** | → 翌日3/21に再電話<br>→ 別の担当者またはフリーダイヤルへ<br>→ 代替返品ルートを検討（販売元の営業担当へ連絡） |

### Day 3: 3/21（金）— FAX/書類送付

| 項目 | 内容 |
|------|------|
| **アクション** | 返品申請書のFAX送信 |
| **確認時刻** | 3/21 夜（FAX送信後） |
| **確認事項** | ✅ FAX送信完了確認<br>✅ 送信控えの保存<br>✅ 返品完了予定日の確認 |
| **想定問題** | ❌ FAXが繋がらない<br>❌ 送信エラーで不達<br>❌ 必要書類が不足 |
| **対処法** | → メール送信に切り替え<br>→ 再送信または電話で確認<br>→ 不足書類を即日準備・再送 |
