## 4. 重症患者割合2割要件の確認

### 計算式
```
重症患者割合 = 重症患者数 ÷ 在宅患者総数 × 100
```

### 判定フロー

```
在宅患者総数: 20名
重症患者数: 4名以上 → 2割以上クリア ✅
重症患者数: 3名以下 → 2割未満 ❌（月1回算定に低下）
```

### 重症患者の定義（別表8の2・8の3）

| カテゴリ | 主な対象 |
|----------|---------|
| がん | 末期悪性腫瘍 |

| 栄養 | 経管栄養・中心静脈栄養 |

## 5. ルート選択フローチャート

```

    │

    │                  │
    │                  └─ 6品目+注射剤1品目 → 達成
    │
    └─ 小児在宅実績あり → ルートD（小児系）
                          │
                          └─ 6回/年以上 → 達成
```

## 6. ゆうすけ向け推奨アクション

### 最優先: 重症患者割合の確認
1. MCS操作で在宅患者総数と重症患者数を正確にカウント
2. 自己評価チェックシート（`zaitaku-sogo-taisei-kazan2-self-assessment.md`）を参照
3. 2割以上なら月2回算定の前提条件クリア

### 次優先: ルート選択
- **ルートA推奨**: 個人宅訪問が中心の場合
  - 目標: 個人宅+緊急訪問 240回/年（月平均20回）
  - 達成見込: 6ヶ月〜1年

- **ルートC検討**: 麻薬備蓄体制が整っている場合
  - 要件: 麻薬6品目 + 注射剤1品目 + 無菌設備
  - 達成見込: 3ヶ月〜半年

### 注意点
- **重症患者割合2割未満の場合**: 月2回算定不可 → 大幅減収リスク
- **早めの対応**: 5月確認時期までに2割以上を確保
