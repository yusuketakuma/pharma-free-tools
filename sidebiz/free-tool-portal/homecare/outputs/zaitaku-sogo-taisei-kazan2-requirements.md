# 在宅薬学総合体制加算2（令和8年6月改定）要件整理

**作成日**: 2026年3月18日
**担当**: 薬剤師エージェント
**情報源**: 厚生労働省告示資料、税理士法人辻総合会計解説、アウトドア薬局解説

---

## 1. 改定の全体像

令和8年6月改定で、在宅薬学総合体制加算は「設備投資によるクリア」から「実績の積み上げと人員配置」重視へ転換。加算2は実績に基づく高度な在宅機能を評価する位置づけ。

---

## 2. 点数比較

| 区分 | 現行 | 改定後 | 備考 |
|------|------|--------|------|
| 加算1 | 15点 | **30点** | 実績要件48回/年に引上げ |
| 加算2（単一建物1人） | 50点 | **100点** | 個人宅訪問を手厚く評価 |
| 加算2（その他） | 50点 | **50点** | 施設在宅は据置き |

---

## 3. 加算1と加算2の要件比較

### 加算1（基本的な在宅体制）

| 要件 | 内容 |
|------|------|
| 在宅実績 | 直近1年間の訪問・指導実績 **48回以上**（旧24回） |
| 24時間体制 | 在宅患者への24時間対応体制 |
| 緊急対応 | 緊急時の調剤・訪問体制整備 |
| 情報連携 | 医療機関・介護施設との連携体制 |

### 加算2（高度な在宅機能）※加算1の全要件を含む

| 要件カテゴリ | 内容 |
|-------------|------|
| **前提** | 加算1の全要件を満たすこと |
| **麻薬・無菌調剤** | 医療用麻薬6品目以上（注射剤1品目以上含む）＋無菌調剤設備、**または** 小児在宅実績6回/年以上 |
| **実績要件（パターンA）** | 個人宅・緊急対応240回/年以上 かつ 全体の2割超 |
| **実績要件（パターンB）** | 個人宅・緊急対応480回/年以上 かつ 全体の1割超 |
| **人員配置** | 原則として開局時間中2名以上の保険薬剤師が常駐 |

### 重要な変更点

- **無菌製剤処理設備の基準が廃止** → 設備単体ではクリアできない
- 麻薬調剤・無菌製剤処理等の「実績」が重視される方向へ
- 常勤換算薬剤師数の基準が追加

---

## 4. 実績に含まれる指導料

以下の算定回数を合算してカウント：
- 在宅患者訪問薬剤管理指導料1
- 在宅患者緊急訪問薬剤管理指導料
- 在宅患者緊急時等共同指導料
- 介護保険における居宅療養管理指導

## 5. 届出に必要な書類・手続き

| 項目 | 内容 |
|------|------|
| 届出様式 | 様式87の3の5（地方厚生局） |
| 届出先 | 管轄の地方厚生局長 |
| 実績報告 | 直近1年間の算定回数実績 |
| 麻薬関連 | 麻薬小売業者免許の写し、麻薬品目リスト |
| 人員関連 | 常勤換算薬剤師数の証明 |
| 設備関連 | 無菌調剤設備の整備状況（該当する場合） |

## 6. 自薬局での対応可能性評価

### チェックリスト

#### A. 加算1要件（前提）

- [ ] 直近1年間の在宅訪問実績48回以上か？ → ワークシートで確認（`zaitaku-sogo-taisei-kazan1-jisseki-worksheet.md`）
- [ ] 24時間対応体制が整備されているか？
- [ ] 緊急時の調剤・訪問体制があるか？
- [ ] 医療機関・介護施設との連携体制があるか？

#### B. 加算2追加要件

- [ ] **麻薬**: 医療用麻薬6品目以上を備蓄しているか？（注射剤1品目以上含む）
- [ ] **無菌調剤**: 無菌室・クリーンベンチ・安全キャビネットのいずれかを備えているか？
- [ ] **小児在宅**: 上記の代替として、小児在宅実績6回/年以上があるか？
- [ ] **実績（パターンA）**: 個人宅訪問＋緊急対応が240回/年以上 かつ 全体の2割超か？
- [ ] **実績（パターンB）**: 個人宅訪問＋緊急対応が480回/年以上 かつ 全体の1割超か？
- [ ] **人員**: 開局時間中2名以上の保険薬剤師が常駐しているか？

#### C. 維持可能性

- [ ] 翌年度も実績要件を維持できる見込みがあるか？
- [ ] 薬剤師の退職・異動リスクは許容範囲か？

## 9. 結論・次アクション

### 要件充足の見通し

加算2は加算1よりも格段にハードルが高い。特に以下がボトルネックになりやすい：

1. **個人宅訪問の実績数**: 240回/年（月20回）以上が必要（パターンA）
2. **薬剤師2名常駐**: 小規模薬局では確保が困難な場合あり
3. **麻薬6品目＋注射剤**: 在宅での麻薬注射対応は準備が必要

### 次ステップ

1. まず加算1の実績48回要件の充足確認を優先（ワークシート準備済み）
2. 加算1取得を確実にした上で、加算2の要件ギャップ分析を実施
3. ゆうすけによるMCS実績データ抽出が必須（個人宅比率の把握）
