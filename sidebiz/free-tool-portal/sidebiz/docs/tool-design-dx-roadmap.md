# 薬局DXロードマップ診断 設計メモ

## 概要
薬局のDX（デジタルトランスフォーメーション）成熟度を診断し、優先すべき改善領域と具体的アクションプランを提示するツール。

## 診断領域（5領域20問）

### 1. 電子化・ペーパーレス（4問）
- 電子薬歴の活用度
- 処方箋電子化の対応状況
- 書類・申請書のデジタル化
- クラウドツールの導入状況

### 2. 自動化・効率化（4問）
- 調剤業務の自動化（PTPシート、散剤分包機等）
- 在庫管理の自動化
- 予約・受付システム
- レセコン・請求処理の効率化

### 3. 患者対応・コミュニケーション（4問）
- オンライン服薬指導の実施状況
- LINE・SMS等での患者連絡
- 服薬リマインダー提供
- 患者向け情報発信（SNS・HP）

### 4. データ活用・分析（4問）
- 売上・処方箋データの分析活用
- 在庫データの可視化
- 患者属性・傾向分析
- KPI設定・モニタリング

### 5. 組織・人材（4問）
- スタッフのデジタルスキル
- DX推進担当の有無
- 研修・学習機会の提供
- 外部サービス・ベンダー連携

## スコアリング

- 各質問: 1〜5点（5段階）
- 領域別スコア: 各4問の合計（4〜20点）
- 総合スコア: 20問の合計（20〜100点）

## レベル分類

| 総合スコア | レベル | 状態 |
|-----------|--------|------|
| 80-100 | 先進 | DX先進薬局・他局のモデルに |
| 60-79 | 進行中 | 主要システム導入済み・最適化段階 |
| 40-59 | 初期 | 基礎的な電子化進行中・優先領域明確化必要 |
| 20-39 | 未着手 | アナログ運用中心・段階的導入計画必要 |

## 結果表示

1. **レーダーチャート**: 5領域の可視化
2. **優先改善領域**: 最もスコアが低い2領域をハイライト
3. **アクションプラン**: レベル別・領域別の具体的アクション提案
4. **ROI試算**: 効率化による時間削減・収益向上の概算

## 収益化導線

- 結果画面に「詳細なDX導入支援はこちら」→ コンサルティング案内
- 「自薬局のDX成熟度を共有」→ SNSシェア機能
- PDF出力機能 → 管理者報告用

## 実装優先度

- **優先度**: 高（2025年問題・かかりつけ薬局化需要）
- **開発難易度**: 中（他診断ツールと同様の構造）
- **想定開発時間**: 1-2時間
