## 本文（有料部分 ¥500）

### 1. 在庫管理の自動化

**使ったツール**: スプレッドシート + ChatGPT

**やったこと**:
1. 既存の在庫Excelをスプレッドシートに移行
2. ChatGPTで発注判定ロジックを作成
3. Apps Scriptで自動通知を設定

**結果**:
- 発注ミスがゼロに
- 発注作業が週2回×30分 → 週1回×10分に短縮
- **週約50分節約**

**コード例**（ChatGPTが書いたもの）:
```javascript
function checkStockAndNotify() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('在庫');
  const data = sheet.getDataRange().getValues();
  
  for (let i = 1; i < data.length; i++) {
    const drugName = data[i][0];
    const currentStock = data[i][1];
    const threshold = data[i][2];
    
    if (currentStock < threshold) {
      MailApp.sendEmail({
        to: 'pharmacy@example.com',
        subject: `【発注通知】${drugName}の在庫が閾値を下回りました`,
        body: `${drugName}: 現在在庫 ${currentStock}、発注推奨数量 ${threshold * 2}`
      });
    }
  }
}
```

### 2. 服薬指導シートの自動生成

**使ったツール**: Claude / ChatGPT

**やったこと**:
1. よくある質問パターンをテンプレート化
2. 薬効・副作用・注意点をAIに整理してもらう
3. 患者向けにわかりやすい言葉に変換

**プロンプト例**:
```
以下の薬について、高齢の患者さん向けに服薬指導シートを作成してください。
- 薬名: [薬品名]
- 効能: [効能]
- 主な副作用: [副作用]
- 注意点: [食事・併用禁忌など]

出力形式:
- 効能（1文で）
- 飲み方（箇条書き3点）
- 注意すること（箇条書き3点）
- 副作用が出たら（1文）
```

**結果**:
- 1枚あたり15分 → 3分に短縮
- 1日平均2枚作成 → **1日24分節約**

### 3. 薬歴入力の効率化

**使ったツール**: 音声入力 + AI要約

**やったこと**:
1. 患者との会話をメモ（または音声録音）
2. AIで「SOAP形式」に整形
3. 電子薬歴にコピペ

**プロンプト例**:
```
以下の患者対応メモを、薬歴用のSOAP形式で整理してください。
メモ: [患者との会話内容]

出力形式:
- S（主観的情報）:
- O（客観的情報）:
- A（アセスメント）:
- P（計画）:
```

**結果**:
- 1件あたり10分 → 5分に短縮
- 1日平均4件 → **1日20分節約**
