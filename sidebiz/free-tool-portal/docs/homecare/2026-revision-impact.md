# 2026年（令和8年度）診療報酬・調剤報酬改定
## 訪問薬剤管理への影響サマリ

**最終更新**: 2026年3月6日 18:35
**情報源**: m3.com, mediva.co.jp, note.com, PMDA, psft.co.jp, medical-unite-lab.com

---

## 1. 訪問診療薬剤師同時指導料【最重要・新設】

### 概要
訪問診療を行う医師と薬局の薬剤師が同時に患家を訪問し、服薬指導を行った場合を評価する新たな点数。

| 項目 | 内容 |
|------|------|
| **点数** | 300点 |
| **算定頻度** | 6ヶ月に1回 |
| **対象** | 在宅患者（医科・調剤双方に評価） |
| **目的** | ポリファーマシー・残薬対策 |

### 実務への影響
- **医師-薬剤師連携の強化**: 同時訪問により処方意図の共有がリアルタイムに
- **残薬確認の効率化**: 患者宅での薬剤確認が医師同席で実施可能
- **服薬管理の質向上**: 多職種での服薬指導が評価される

### 準備すべきこと
1. 連携している訪問診療医師との同時訪問スケジュール調整
2. 同時訪問時の服薬指導プロトコル策定
3. 対象患者の選定基準（ポリファーマシー・残薬リスク）

---

## 2. 在宅医療充実体制加算【新設】

### 概要
従来の「在宅緩和ケア充実診療所・病院加算」が改編され、地域で重症患者への対応や看取りを積極的に担う医療機関をより高く評価。

| 項目 | 内容 |
|------|------|
| **点数** | 800点（単一建物患者1人の場合） |
| **対象** | 重症患者対応・看取りを積極的に担う医療機関 |

### 実務への影響
- 連携医療機関が本加算を取得している場合、訪問薬剤管理の需要増の可能性
- 重症患者への訪問頻度・質が求められる

---

## 3. 重症患者割合20%要件【新規】

### 概要
「在宅時医学総合管理料」等の月2回以上訪問の区分において、新たな算定要件が追加。

| 項目 | 内容 |
|------|------|
| **要件** | 訪問患者のうち重症患者割合が20%以上 |
| **影響** | 要介護度の低い患者への一律的な頻回訪問見直し |

### 実務への影響
- 軽症患者中心の頻回訪問は評価が下がる方向
- 重症患者の受入体制強化が必要
- **今後の改定で基準が引き上げられる可能性あり**

---

## 4. 調剤報酬改定（調剤薬局関連）

### 廃止・統合
- **かかりつけ薬剤師指導料（包括管理料）**: 廃止

### 新設
| 項目 | 内容 |
|------|------|
| **地域支援・医薬品供給対応体制加算** | 旧地域支援体制加算/旧後発医薬品調剤体制加算の統合 |
| **調剤ベースアップ評価料** | R8.6月〜4点/処方箋、R9.6月〜8点 |
| **物価対応料** | 物価高騰対応 |

### 実務への影響
- かかりつけ薬剤師制度の見直しに伴う指導体制の再構築
- 地域支援・医薬品供給対応体制加算の算定要件確認

---

## 4.1 かかりつけ薬剤師関連の新設加算【2026年3月6日追記】

### 新設加算一覧
| 加算名 | 点数 | 内容 |
|--------|------|------|
| **かかりつけ薬剤師訪問加算** | 230点 | 求めに応じた患家への訪問、残薬整理・指導を評価 |
| **かかりつけ薬剤師フォローアップ加算** | 50点 | 電話等での継続的な服薬状況・残薬状況の確認を評価 |

### 制度変更のポイント
- **かかりつけ薬剤師指導料（76点）**: 独立項目としては廃止
- **服薬管理指導料1のイ**: かかりつけ薬剤師による指導として再設計
- **実績ベース評価への移行**: 同意取得だけでなく、具体的な介入実績を重視

### 実務への影響
- **訪問指導の価値向上**: 230点の新設加算により、在宅訪問の収益性改善
- **電話フォローの評価**: 50点加算で、電話対応も収益化可能
- **対人業務の質が問われる**: 具体的な介入実績（残薬整理・服薬指導）が必須

## 5. アクションアイテム

### 即時対応（〜R8.6月）
- [ ] 訪問診療薬剤師同時指導料の算定要件詳細確認
- [ ] 連携訪問診療医師との同時訪問調整
- [ ] 対象患者リスト作成（ポリファーマシー・残薬リスク）
- [ ] かかりつけ薬剤師訪問加算（230点）の対象患者選定
- [ ] かかりつけ薬剤師フォローアップ加算（50点）の運用フロー策定

### 中期対応（R8.6月〜R9.6月）
- [ ] 重症患者割合20%要件の達成状況確認
- [ ] 地域支援・医薬品供給対応体制加算の算定状況確認

### 長期対応（R9.6月以降）
- [ ] 重症患者割合基準の引き上げに備えた受入体制強化
