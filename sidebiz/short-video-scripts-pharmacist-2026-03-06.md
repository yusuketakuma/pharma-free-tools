## 台本1: 風邪薬の飲み合わせ【危険】

**フック（0-3秒）**
「風邪薬、これと一緒に飲んでませんか？」

**本編（3-45秒）**
「薬剤師が教えます。
風邪薬と一緒に飲むと危険な組み合わせ3選！

1つ目：風邪薬 + 眠気止め
→ 眠気成分が重複して、強い眠気や集中力低下

2つ目：風邪薬 + 鎮痛剤
→ アセトアミノフェンが重複して、肝臓に負担

3つ目：風邪薬 + アルコール
→ 眠気成分が増強されて、危険なレベルに

チェック方法は簡単。
薬の箱の「成分」を見て、同じ名前がないか確認！」

**CTA（45-60秒）**
「他の飲み合わせも知りたい？
プロフィールのリンクから詳細記事を見てね」

## 台本2: 薬の保管場所【意外】

**フック（0-3秒）**
「薬、お風呂場に置いてませんか？」

**本編（3-45秒）**
「薬剤師が教えます。
薬の保管場所、間違ってる人多いんです！

NGな場所3選：
1. お風呂場 → 湿気で成分が変質
2. キッチン → 熱と湿気で品質低下
3. 車の中 → 温度変化で効果減退

正しい保管場所：
- 直射日光が当たらない
- 湿気が少ない
- 温度が安定している
→ closet（クローゼット）や引き出しの中がベスト！」

**CTA（45-60秒）**
「薬の正しい管理方法、もっと知りたい？
フォローしてね！」

## 台本4: 睡眠薬の真実【誤解】

**フック（0-3秒）**
「睡眠薬、飲み続けると依存するって本当？」

**本編（3-45秒）**
「薬剤師が解説します。
睡眠薬についてる5つの誤解！

誤解1：依存する
→ 新しい薬は依存性が低い設計

誤解2：飲み続けないと眠れない
→ 徐々に減らせます

誤解3：頭がおかしくなる
→ 適正使用なら安全

誤解4：自然な睡眠じゃない
→ 睡眠の質を改善します

誤解5：飲まない方がいい
→ 睡眠不足の方が体に悪影響

大事なのは、医師と薬剤師に相談すること！」

**CTA（45-60秒）**
「睡眠の質を上げる方法、気になる？
コメントで教えて！」

薬の使用期限、意外と知らないルール！

ポイントは2つ：
1. 「使用期限」の表示があるか
2. 開封後の期間

目安：
