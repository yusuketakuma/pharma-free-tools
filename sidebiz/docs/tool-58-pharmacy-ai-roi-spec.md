# 58ツール目: 薬局AI導入ROI診断 - 仕様書

## 概要
薬局におけるAI導入の投資収益率（ROI）を診断する無料ツール。導入コスト・削減効果・投資回収期間を可視化し、経営層への投資判断支援を行う。

## ターゲット
- 薬局経営者・管理者
- DX推進担当薬剤師
- AI導入を検討している薬局

## 機能要件

### 1. AI導入シナリオ選択
- **薬歴入力支援AI**: 薬歴作成時間の短縮
- **疑義照会支援AI**: 疑義照会メール・電話の効率化
- **在庫管理AI**: 発注・在庫管理の自動化
- **患者対応AI**: チャットボット・自動応答

### 2. 現状入力
- 処方箋枚数/月
- 薬剤師人数
- 現状の業務時間（薬歴入力・疑義照会・在庫管理）
- 人件費（時給・月給）

### 3. AI導入コスト入力
- 初期費用（ライセンス・導入支援）
- 月額費用
- トレーニング費用

### 4. 効果計算
- 時間短縮効果（時間/月）
- コスト削減効果（円/月）
- 収益向上効果（円/月）

### 5. 結果表示
- 投資回収期間（ヶ月）
- 3年間のROI（%）
- 推奨アクション

## RICE Score
- **Reach**: 8,000（薬局AI導入に関心のある薬剤師）
- **Impact**: 3（収益化に直結）
- **Confidence**: 0.7（市場調査で需要確認済み）
- **Effort**: 2（4時間）
- **RICE Score**: 8,000 × 3 × 0.7 / 2 = **8,400**
