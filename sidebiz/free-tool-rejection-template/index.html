返戣理由別回答テンプレート

1. 処方日から受領日まで4日を超える場合
【回答書】 返戣理由：処方日から受領日まで4日を超えている 回答内容： 当該処方箋について、以下の事情により4日を超えての受領となりました。 【具体的な事情】 ・患者様の都合により来局が遅れました ・処方箋発行日：○年○月○日 ・来局日：○年○月○日 ・遅延理由：[具体的理由を記載] 【添付資料】 ・処方箋の写し ・患者様からの事情聴取メモ 以上、ご査収のほどよろしくお願い申し上げます。
2. 疑義照会後の処方内容不一致
【回答書】 返戣理由：疑義照会後の処方内容と調剤内容が不一致 回答内容： 当該処方箋について、以下の通り疑義照会を実施し、医師の指示に基づき調剤いたしました。 【疑義照会内容】 ・照会日時：○年○月○日 ・照会医師：○○医院 ○○先生 ・照会内容：[具体的な疑義点] ・医師指示：[医師からの回答内容] 【調剤内容との整合性】 疑義照会の結果、医師指示に基づき以下の通り調剤いたしました。 ・調剤内容：[具体的な調剤内容] 【添付資料】 ・疑義照会記録の写し ・調剤録の該当部分 以上、ご査収のほどよろしくお願い申し上げます。
3. 用法・用量の不適切な記載
【回答書】 返戣理由：用法・用量の記載が不適切 回答内容： 当該処方箋の用法・用量について、以下の通り説明申し上げます。 【処方内容】 ・薬剤名：○○ ・処方用量：[用量] ・処方用法：[用法] 【調剤時の確認事項】 ・用量確認：[確認内容] ・用法確認：[確認内容] ・患者様への説明：[説明内容] 【特記事項】 [特別な事情があれば記載] 【添付資料】 ・調剤録の写し ・服薬指導記録 以上、ご査収のほどよろしくお願い申し上げます。
4. 剤形・規格の不一致
【回答書】 返戣理由：剤形・規格が不一致 回答内容： 当該処方箋について、以下の事情により剤形・規格の変更を行いました。 【変更内容】 ・処方：[処方された剤形・規格] ・調剤：[実際に調剤した剤形・規格] 【変更理由】 ・在庫状況：[在庫の有無] ・患者様の状態：[服用しやすさ等の配慮] ・医師への確認：[確認の有無と内容] 【添付資料】 ・調剤録の写し ・在庫確認記録（該当する場合） 以上、ご査収のほどよろしくお願い申し上げます。
5. 後発品への変更に関する返戻
【回答書】 返戣理由：後発品への変更が適切でない 回答内容： 当該処方箋の後発品変更について、以下の通り回答申し上げます。 【処方内容】 ・先発品名：○○ ・後発品名：○○ 【変更の妥当性】 ・患者様への説明：変更の同意を得ました ・医師の指示：[変更可否の指示内容] ・薬学的同等性：[同等性の確認内容] 【患者様の状況】 ・変更理由の理解：確認済み ・服用上の注意：説明済み 【添付資料】 ・調剤録の写し ・服薬指導記録 以上、ご査収のほどよろしくお願い申し上げます。
6. 在宅医療関連の返戣
【回答書】 返戣理由：在宅患者指導料等の算定に関する疑義 回答内容： 当該在宅医療の指導について、以下の通り回答申し上げます。 【実施内容】 ・訪問日：○年○月○日 ・訪問時間：○時○分〜○時○分（○分間） ・指導内容：[具体的な指導内容] 【患者様の状況】 ・主な疾患：[疾患名] ・服薬状況：[服薬状況の説明] ・指導の必要性：[なぜ指導が必要か] 【算定の根拠】 ・[算定要件を満たしている理由] 【添付資料】 ・訪問記録の写し ・指導記録 以上、ご査収のほどよろしくお願い申し上げます。
7. 重複投薬期間の返戣
【回答書】 返戣理由：重複投薬期間がある 回答内容： 当該処方箋について、以下の通り重複投薬期間の妥当性を説明申し上げます。 【重複期間の内容】 ・前回処方期間：○年○月○日〜○年○月○日 ・今回処方期間：○年○月○日〜○年○月○日 ・重複日数：○日間 【重複の理由】 ・[具体的な理由：患者の旅行、予備確保等] 【薬学的確認事項】 ・相互作用の有無：確認済み ・服薬スケジュール：患者様に説明済み 【添付資料】 ・前回調剤録の写し ・今回調剤録の写し 以上、ご査収のほどよろしくお願い申し上げます。
8. 保険者情報の不備
【回答書】 返戣理由：被保険者情報等の不備 回答内容： 当該処方箋の保険情報について、以下の通り回答申し上げます。 【確認内容】 ・保険種別：[健康保険/国民健康保険/後期高齢者等] ・被保険者番号：[番号] ・有効期限：[期限] 【不備の内容と対応】 ・[不備の具体的な内容] ・[実施した確認・対応] 【添付資料】 ・保険証の写し（必要に応じて） 以上、ご査収のほどよろしくお願い申し上げます。

返戣対応・再請求のコツ

📌 返戣理由の分類
【技術的返戣】 ・用法・用量の不備 ・剤形・規格の不一致 ・重複投薬期間 → 調剤時の確認強化で予防可能 【事務的返戣】 ・保険者情報の不備 ・処方日から4日超過 ・記載漏れ → 受付時の確認強化で予防可能 【その他】 ・在宅医療関連 ・特殊な調剤 → 算定要件の再確認で予防可能
📝 回答書作成のポイント
1. **事実関係を明確に** - いつ・何を・どのように確認したか - 数値・日付は正確に記載 2. **根拠を示す** - なぜその対応が妥当だったか - 薬学的判断の根拠 3. **添付資料を過不足なく** - 必要な資料は全て添付 - 不必要な資料は添付しない 4. **期限内提出を徹底** - 通常30日以内 - 余裕を持って作成開始
🔄 再発防止策の実施
【調剤時】 ・ダブルチェックの徹底 ・疑義照会の記録保存 ・服薬指導記録の整備 【受付時】 ・処方箋の有効期限確認 ・保険証の確認 ・患者情報の更新 【事務処理】 ・請求データの確認 ・レセプト点検の強化 ・定期的な傾向分析
📊 返戣率改善のKPI
【監視指標】 ・月間返戣件数 ・返戣理由別件数 ・再請求成功率 ・平均対応日数 【目標値】 ・返戣率：1%以下 ・再請求成功率：80%以上 ・平均対応日数：7日以内 【改善サイクル】 1. 月次で返戣傾向を分析 2. 多発理由の対策を検討 3. 対策実施・効果測定 4. 次月の傾向と比較
