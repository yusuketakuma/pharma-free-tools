# 薬剤師のためのChatGPT/Claude業務効率化プロンプト集

## はじめに

このプロンプト集は、調剤薬局で働く薬剤師がChatGPTやClaudeを使って業務を効率化するためのテンプレート集です。

薬歴入力、服薬指導、疑義照会、在宅医療など、日常業務で即座に使えるプロンプトを厳選しました。

**対象読者**:
- 調剤薬局で働く薬剤師
- 在宅医療・訪問薬剤管理に携わる薬剤師
- 業務効率化をしたい薬剤師

---

## 使い方

1. 使いたいプロンプトをコピー
2. `[ ]`で囲まれた部分を実際の情報に書き換え
3. ChatGPTまたはClaudeに貼り付け
4. 出力を確認・修正して使用

**注意**:
- 個人情報は入力しない（イニシャルや年齢のみ）
- 出力は必ず自分で確認する
- 薬剤師の判断を置き換えるものではない

## セクション1: 薬歴入力の効率化

### プロンプト1-1: 服薬指導内容の薬歴形式への変換

```
あなたは調剤薬局で働く経験豊富な薬剤師です。
以下の服薬指導の内容を、薬歴の形式でまとめてください。

【患者情報】
- 年齢: [年齢]
- 主病名: [病名]
- 処方薬: [薬剤名] [用量] [用法]

【服薬指導で聞き取った内容】
[会話内容を箇条書きで記載]

【出力形式】
以下の形式で薬歴を作成してください：
1. S（主観的情報）
2. O（客観的情報）
3. A（アセスメント）
4. P（計画）

簡潔かつ必要十分な情報を含めてください。
```

### プロンプト1-2: 薬歴の要約と重点ポイント抽出

```
以下の薬歴から、次回来局時のフォローアップに必要な情報を抽出してください。

【過去の薬歴】
[薬歴テキストを貼り付け]

【出力】
- 重要な副作用歴・アレルギー:
- 継続中の指導事項:
- 次回確認すべきポイント:
- 患者の特徴（服薬コンプライアンス、理解度など）:
```

## セクション2: 疑義照会の準備

```
【処方内容】

【出力】

4. 患者背景との不整合

```

### プロンプト2-2: 疑義照会の問い合わせ文面作成

```

[問題点を記載]

【出力】
1. 開業の挨拶（1文）
2. 患者の特定情報（氏名、年齢、保険証番号の下4桁）
3. 処方内容の確認（薬剤名、用量）
4. 疑義の内容（簡潔に）
5. 確認事項（はい/いいえで答えられる形式）
6. まとめ（確認内容の再確認）
```

```
【出力】
```

## ボーナスセクション: プロンプトのカスタマイズ方法

### 基本的なカスタマイズのポイント

1. **患者背景の追加**: 年齢、腎機能、アレルギー歴などを追加することで精度向上
2. **出力形式の指定**: 箇条書き、表形式、文字数制限などを指定
3. **役割の明確化**: 「経験豊富な薬剤師として」「患者にわかりやすく」など
4. **制約の追加**: 「専門用語を避ける」「3分で読める量にする」など

### プロンプト改善のテンプレート

```
以下のプロンプトを改善してください。

【元のプロンプト】
[改善したいプロンプト]

【改善の目的】
[より具体的にしたい/より簡潔にしたい/出力形式を変えたい]

【出力】
改善後のプロンプトと、改善のポイント
```

## 注意事項

### 個人情報の取り扱い
- 氏名はイニシャルまたは「Aさん」等の仮名を使用
- 生年月日ではなく年齢のみを使用
- 住所・電話番号等は入力しない

### 出力の確認
- AIの出力は必ず自分で確認する
- 特に用量・用法は必ず照合する
- 医学的判断は薬剤師自身が行う

### ツールの使い分け
- **ChatGPT**: 一般的な質問、文書作成
- **Claude**: 長文の要約、日本語の自然さ重視
