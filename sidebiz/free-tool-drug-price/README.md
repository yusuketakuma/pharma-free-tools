# 後発品切り替えシミュレーター

薬剤師が患者さんに後発品への切り替えを説明する際に使える、無料の薬価差額計算ツールです。

## 特徴

- 📱 **モバイルファースト**: スマホで使いやすいUI
- 🇯🇵 **日本語専用**: 薬価差額を日本の保険制度に合わせて計算
- 📋 **コピー機能**: 患者さん向け説明文をワンクリックでコピー
- 🔒 **完全オフライン**: データは端末内のみ、プライバシー保護

## 使い方

1. 薬品名を検索・選択
2. 後発品を選択
3. 日数を入力
4. 薬価差額（1日・期間・年間）を確認
5. 「説明文をコピー」で患者さんへの説明に活用

## 収益モデル

このツールは無料で提供しています。もし役立った場合は、以下の有料製品もご検討ください：

- **薬剤師向けAIプロンプト集**: 業務効率化のためのChatGPT/Claudeプロンプト集
- **Notion薬剤管理テンプレート**: 訪問薬剤管理・在庫管理のためのテンプレート
