# 薬剤師のためのChatGPT/Claude業務効率化プロンプト集

## はじめに

このプロンプト集は、調剤薬局で働く薬剤師がChatGPTやClaudeを使って業務を効率化するためのテンプレート集です。
薬歴入力、服薬指導、疑義照会など、日常業務で即座に使えるプロンプトを厳選しました。

---

## セクション1: 薬歴入力の効率化

### プロンプト1-1: 服薬指導内容の薬歴形式への変換

```
あなたは調剤薬局で働く経験豊富な薬剤師です。
以下の服薬指導の内容を、薬歴の形式でまとめてください。

【患者情報】
- 年齢: [年齢]
- 主病名: [病名]
- 処方薬: [薬剤名] [用量] [用法]

【服薬指導で聞き取った内容】
[会話内容を箇条書きで記載]

【出力形式】
以下の形式で薬歴を作成してください：
1. S（主観的情報）
2. O（客観的情報）
3. A（アセスメント）
4. P（計画）

簡潔かつ必要十分な情報を含めてください。
```

### プロンプト1-2: 薬歴の要約と重点ポイント抽出

```
以下の薬歴から、次回来局時のフォローアップに必要な情報を抽出してください。

【過去の薬歴】
[薬歴テキストを貼り付け]

【出力】
- 重要な副作用歴・アレルギー:
- 継続中の指導事項:
- 次回確認すべきポイント:
- 患者の特徴（服薬コンプライアンス、理解度など）:
```

## セクション2: 疑義照会の準備

### プロンプト2-1: 処方箋の矛盾・リスクの抽出

```

## 付録: プロンプトのカスタマイズ方法

### 基本的なカスタマイズのポイント

1. **患者背景の追加**: 年齢、腎機能、アレルギー歴などを追加することで精度向上
2. **出力形式の指定**: 箇条書き、表形式、文字数制限などを指定
3. **役割の明確化**: 「経験豊富な薬剤師として」「患者にわかりやすく」など

### 注意点

- 個人情報は入力しない（イニシャルや年齢のみにとどめる）
- 出力は必ず確認する（AIの誤りを自分でチェックする）
- 薬剤師の判断を置き換えるものではない（あくまで補助ツール）

## セクション6: 調剤実務の効率化

### プロンプト6-1: 調剤エラー防止チェックリスト作成

```
以下の処方箋について、調剤時に確認すべきチェックリストを作成してください。

【処方内容】
[薬剤名] [用量] [用法] [日数]
[薬剤名] [用量] [用法] [日数]
...

【出力】
1. 薬剤ごとの確認事項（剤形、規格、メーカー）
2. 類似名薬剤との混同リスク
3. 用量の妥当性チェック
4. 併用薬との相互作用チェック
5. 患者固有の注意点（アレルギー、過去歴）

簡潔なチェックリスト形式で出力してください。
```

### プロンプト6-2: 薬局スタッフへの引継ぎメモ作成

```
複雑な処方箋・患者対応について、次のシフト担当者への引継ぎメモを作成してください。

【対応内容】
[何があったかを記載]

【今後の対応が必要な事項】
[継続して確認すべきことを記載]

【出力】
1. 要点（1-2文で状況を要約）
2. 経緯（箇条書きで時系列）
3. 残課題（優先度付き）
4. 担当者への依頼事項
5. 関連資料の場所

簡潔かつ必要十分な情報で、30秒で読める分量にしてください。
```

## セクション7: 学習・自己研鑽

### プロンプト7-1: 新薬クイックリファレンス作成

```
[薬剤名]について、薬局実務ですぐに使えるクイックリファレンスを作成してください。

【出力】
1. 基本情報（一般名、薬効分類、適応）
2. 既存薬との違い（メリット・デメリット）
3. 調剤時の注意（剤形、保管、安定性）
4. 服薬指導の要点（3-5項目）
5. 疑義照会すべきケース

1枚のチラシ（A4）に収まる分量で、実務重視の内容にしてください。
```

### プロンプト7-2: 症例学習の振り返りレポート

```
以下の症例について、学習レポートを作成してください。

【症例概要】
- 患者背景: [年齢、性別、主病名]

- 疑義照会内容: [あれば記載]

【出力】
1. 症例の要約（何が起きたか）

## セクション8: コミュニケーション・接遇

### プロンプト8-1: クレーム対応の振り返り

```
患者からのクレーム対応について、改善のための振り返りを行ってください。

【状況】
- クレーム内容: [何を言われたか]
- 当時の対応: [どう対応したか]
- 結果: [どうなったか]

【出力】
1. クレームの背景要因分析
2. 当時の対応の評価（良かった点、改善点）
3. より良い対応案
4. 再発防止策（システム的、個人的）
5. 心理的対処法（自分へのメッセージ）

建設的な振り返りになるよう、感情と事実を分けて分析してください。
```

```

【連携内容】

- 目的: [情報共有/照会/依頼]
- 内容: [伝えたいことを記載]

【出力】
1. 件名（用件が一目でわかる形式）

```
