# 看護師のためのChatGPT/Claude業務効率化プロンプト集

## はじめに

このプロンプト集は、病院・クリニック・訪問看護で働く看護師がChatGPTやClaudeを使って業務を効率化するためのテンプレート集です。

ケアプラン、看護記録、インシデントレポート、患者教育など、日常業務で即座に使えるプロンプトを厳選しました。

**対象読者**:
- 病院・クリニックで働く看護師
- 訪問看護師
- 看護管理者
- 業務効率化をしたい看護師

---

## 使い方

1. 使いたいプロンプトをコピー
2. `[ ]`で囲まれた部分を実際の情報に書き換え
3. ChatGPTまたはClaudeに貼り付け
4. 出力を確認・修正して使用

**注意**:
- 個人情報は入力しない（イニシャルや年齢のみ）
- 出力は必ず自分で確認する
- 看護師の判断を置き換えるものではない

## セクション1: 看護記録の効率化

### プロンプト1-1: バイタルサインの傾向分析

```
あなたは経験豊富な看護師です。
以下のバイタルサインのデータを分析し、傾向と注意点をまとめてください。

【患者情報】
- 年齢: [年齢]
- 主病名: [病名]
- 現在の治療: [治療内容]

【バイタルデータ】
| 日時 | 体温 | 血圧 | 脈拍 | SpO2 | 呼吸数 |
|------|------|------|------|------|--------|
| [日時] | [値] | [値] | [値] | [値] | [値] |
...

【出力】
1. 各項目の傾向（上昇/安定/低下）
2. 異常値の有無とその解釈
3. 次回観察の重点ポイント
4. 医師への報告必要性の判断
```

### プロンプト1-2: 看護記録のSOAP形式への変換

```
以下の観察内容をSOAP形式の看護記録に変換してください。

【患者情報】
- 年齢: [年齢]
- 主病名: [病名]
- 日時: [日時]

【観察内容】
[観察した内容を箇条書きで記載]

【出力】
以下の形式で記録を作成：
- S（主観的情報）: 患者の訴え
- O（客観的情報）: 観察データ
- A（アセスメント）: 看護診断・問題の分析
- P（計画）: 今後のケア方針

簡潔かつ必要十分な情報を含めてください。
```

## ボーナスセクション: プロンプトのカスタマイズ方法

### 基本的なカスタマイズのポイント

1. **患者背景の追加**: 年齢、ADL、認知機能などを追加して精度向上
2. **出力形式の指定**: 箇条書き、表形式、文字数制限を指定
3. **役割の明確化**: 「経験豊富な看護師として」「患者にわかりやすく」など
4. **制約の追加**: 「専門用語を避ける」「3分で読める量にする」など

### プロンプト改善のテンプレート

```
以下のプロンプトを改善してください。

【元のプロンプト】
[改善したいプロンプト]

【改善の目的】
[より具体的にしたい/より簡潔にしたい/出力形式を変えたい]

【出力】
改善後のプロンプトと、改善のポイント
```

## 注意事項

### 個人情報の取り扱い
- 氏名はイニシャルまたは「Aさん」等の仮名を使用
- 生年月日ではなく年齢のみを使用
- 住所・電話番号等は入力しない

### 出力の確認
- AIの出力は必ず自分で確認する
- 特にバイタルサイン・薬剤情報は必ず照合する
- 医学的判断は看護師自身が行う

### ツールの使い分け
- **ChatGPT**: 一般的な質問、文書作成
- **Claude**: 長文の要約、日本語の自然さ重視
