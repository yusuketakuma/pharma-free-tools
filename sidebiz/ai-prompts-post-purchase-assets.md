## 4. よくある質問（購入者向け）

### Q1: PDFがダウンロードできません
**A:** 
- ブラウザのポップアップブロックを無効にしてください
- 別のブラウザ（Chrome推奨）でお試しください
- ダウンロードリンクの有効期限は購入後30日間です
- それでも解決しない場合は support@example.com まで

### Q2: ChatGPT無料版でも使えますか？
**A:** 
- はい、全てのプロンプトはChatGPT無料版（GPT-3.5）で動作確認済みです
- GPT-4の方がより精密な回答が得られる場合があります
- Claude（無料版）でも同様にご利用いただけます

### Q3: プロンプトをカスタマイズしてもいいですか？
**A:** 
- はい、ぜひカスタマイズしてください
- [ ] の部分を自分の状況に合わせて書き換えることを推奨します
- カスタマイズ版の再配布はご遠慮ください

### Q4: アップデートはありますか？
**A:** 
- はい、随時アップデートを行っています
- 購入者は無償で最新版をダウンロードできます
- アップデート通知はメールでお知らせします

### Q5: 返金してもらえますか？
**A:** 
- はい、購入後7日以内であれば全額返金いたします
- support@example.com まで「返金希望」と明記してご連絡ください
- 返金理由は任意ですが、改善のためにフィードバックをいただけますと幸いです

### Q6: プロンプトがうまく動きません
**A:** 
- [ ] の部分を全て書き換えましたか？
- 入力情報に矛盾がありませんか？
- より具体的な情報を追加してみてください
- それでも解決しない場合は、具体例とともにサポートまでご相談ください

### Q7: 複数のデバイスで使えますか？
**A:** 
- はい、ダウンロードしたPDFは複数デバイスでご利用いただけます
- 個人利用の範囲内でご使用ください

## 5. 返金・サポート対応フロー

### 返金リクエスト受信時

1. **確認**
   - 注文日が7日以内か確認
   - 注文ID・購入者メールアドレスを確認

2. **対応**
   - 7日以内 → Payhip管理画面から返金処理
   - 8日以降 → 「7日以内の返金ポリシー」を説明し、丁重に断る

3. **返信テンプレート**
```
[お客様名] 様

ご連絡ありがとうございます。

ご注文（注文ID: [ORDER_ID]）の返金処理を承りました。
通常3〜5営業日以内に、お支払い方法に応じて返金されます。

この度はご期待に沿えず、申し訳ございませんでした。
ご意見・ご要望がございましたら、今後の改善のために
お聞かせいただけますと幸いです。

またのご利用を心よりお待ちしております。

薬剤師のためのAIプロンプト集
サポートチーム
```

### サポート問い合わせ受信時

1. **分類**
   - 技術的な問題（ダウンロードエラー等）
   - 使い方の質問
   - コンテンツへの要望
   - その他

2. **対応目安**
   - 技術的問題: 24時間以内
   - 使い方の質問: 48時間以内
   - その他: 72時間以内

3. **エスカレーション**
   - 解決困難な問題は専用メモに記録
   - パターン化した問題はFAQに追加
