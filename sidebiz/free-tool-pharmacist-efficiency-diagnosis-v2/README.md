# 薬剤師業務効率化診断 v2

## 概要

薬剤師の業務効率化レベルを4領域で診断し、スコア化・改善提案を行う無料ツール。

## 機能

- **4領域の診断**: 薬歴入力・疑義照会・在庫管理・服薬指導
- **スコアリング**: 各20点満点で合計80点
- **改善提案**: 低スコア領域への具体的アクション提示
- **収益化CTA**: AIプロンプト集・Notionテンプレートへの導線

## 使い方

1. 8つの質問に回答
2. 「診断結果を見る」をクリック
3. スコアと改善提案を確認
4. 必要に応じてCTAから有料製品へ

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- Schema.org JSON-LD
- GA4イベント計測
