# テーマ抽出レポート — 2026-03-24

## 結論
今回の再調査でも、**最優先は「診断止まりの高需要テーマを、その場で文面・判断材料・次アクションまで出せる実務ツールへ寄せること」**だった。

結論として、トップ3は前回から大きく崩さず以下を維持する。

1. 薬歴下書き・要点整理支援
2. 供給障害患者対応ワークベンチ
3. 返戻再請求ナビ

一方で新しい差分として、**疑義照会・トレーシングレポート文面支援**を強い候補として追加した。薬剤師起点の困りごととして頻度・痛み・HTML適性が高く、既存サイト内に“専用の一軍ツール”がないため、4位に入れる価値がある。

また、**長期収載品選定療養の患者説明**も継続監視対象として追加した。ただし現時点ではトップ3に食い込むほどではなく、優先度は中位。

## 優先候補トップ3

### 1) 薬歴下書き・要点整理支援
- 誰が困っているか: 薬局薬剤師
- 課題場面: SOAPのA/Pで止まる、指導後に記録が残る、書き出しに迷う
- 解決案: 患者属性・処方変更・副作用・生活背景を入れると、SOAP要点・次回確認事項・患者説明メモを即出力
- 新規作成か既存改善か: **既存改善**
- 実現可能性: **高い**

### 2) 供給障害患者対応ワークベンチ
- 誰が困っているか: 薬局薬剤師、管理薬剤師
- 課題場面: 限定出荷時に患者説明、医師連絡、代替案整理、薬歴記録が一気に発生する
- 解決案: 薬剤名・供給状態・患者リスク・残薬・在庫状況から、患者説明文、医師連絡文、記録文、対応チェックリストを出す
- 新規作成か既存改善か: **既存改善**
- 実現可能性: **高い**

### 3) 返戻再請求ナビ
- 誰が困っているか: 薬局事務、請求担当、管理薬剤師
- 課題場面: 返戻理由ごとの処理が迷う、オンライン返戻再請求で返戻ファイル前提の運用が分かりにくい
- 解決案: 返戻理由・提出状態・修正内容を入れると、再請求手順、修正チェック、記録ポイントを出す
- 新規作成か既存改善か: **既存改善**
- 実現可能性: **高い**
