### 🥇 候補1: ECモール出品管理番号自動取得ツール

- **期限**: 2026-04-07
- **報酬**: 5,000円（実績作り目的）

#### 提案文（そのまま送信可）

```
ご依頼内容を拝見し、自信を持ってお手伝いできます。

【対応可能な理由】
Python + BeautifulSoup/Seleniumによるスクレイピング開発を多数手がけており、
Googleスプレッドシートへの自動出力（gspread API連携）の実績も豊富です。

具体的な対応イメージ：
① Pythonスクリプトで3モール（ヤフオク・ラクマ・vestiairecollective）にログイン
② 出品中商品一覧ページから管理番号を抽出
③ Googleスプレッドシートに一覧出力
④ 定期実行対応（必要であればGASでスケジューリングも可能）

【納期目安】
仕様確認（動画拝見）後、3〜5日以内に納品可能です。

【納品物】
- Pythonスクリプト一式（実行環境構築手順付き）
- Googleスプレッドシート連携設定
- 操作マニュアル（初心者にも分かりやすく）
- 初期不具合修正対応（納品後7日間）

【応募時の確認事項】
- 各モールへのログイン情報はお客様ご提供でよいか
- スプレッドシートは既存のものに追記か、新規作成か

オンラインミーティングも可能です。ご検討よろしくお願いいたします。
```

## Step 4: 登録後のやること

- [ ] 本人確認完了を待つ（通常即日〜翌日）

## 注意事項

- 実績ゼロの状態では最初の1件受注が最重要
- 候補1の5,000円は低単価だが、レビュー獲得で以降の単価向上に直結
- 応募は早い者が有利。登録完了後すぐに送信する
- 提案文は案件ごとに少しカスタマイズすると採用率が上がる
